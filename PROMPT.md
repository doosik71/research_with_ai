# PROMPT 예시

## 처음 시작시

```text
`README.md` 파일을 읽고 프로젝트의 성격을 이해하라.
`TOPICS.md` 파일을 읽고 연구 분야와 관련 폴더의 경로를 파악하라.
`RULES.md` 파일을 읽고 문서 작성 규칙과 보고서 작성 규칙 등을 숙지하라.

```

## 논문 목록 업데이트

```text
ㅇㅇㅇ 분야의 논문을 arxiv에서 검색하고 ㅇㅇ 폴더의 `paper_list.md` 파일에 논문 목록을 업데이트하라.
```

## 논문 상세 분석 보고서 작성

```text
이제부터 ㅇㅇㅇ 폴더에서 작업하겠다.
ㅇㅇㅇ 논문의 상세 분석 보고서를 작성하라.
작성된 논문의 상세 분석 보고서가 규칙에 맞는 검토하고 오류를 수정하라.
```

## 문서 형식 후처리

```text
ㅇㅇㅇ 문서의 형식을 검토(markdownlint)하고 오류를 수정하라.
수식의 분별자("$", "$$")도 규칙에 맞게 작성되었는지 확인하고 수정하라.
```

## 커밋

```text
git_commit_message.txt 파일을 작성하라.
```
