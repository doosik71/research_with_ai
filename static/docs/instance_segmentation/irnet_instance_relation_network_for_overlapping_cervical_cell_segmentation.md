# IRNet: Instance Relation Network for Overlapping Cervical Cell Segmentation

- **저자**: Yanning Zhou, Hao Chen (✉), Jiaqi Xu, Qi Dou, Pheng-Ann Heng
- **발표연도**: 2019
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1908.06623

## 1. 논문 개요

이 논문은 세포 덩어리 내 반투명 세포질의 폐색(occlusion)으로 인해 어려움을 겪는 Pap smear 이미지에서의 세포 인스턴스 분할(cell instance segmentation) 문제를 해결하는 것을 목표로 한다. 기존 방법들은 핵(nuclei) 검출 결과에 크게 의존하며 다양한 이물질에 의해 쉽게 방해받는다. 본 논문은 인스턴스 관계 상호작용(instance relation interaction)을 탐색하여 중첩되는 세포 분할을 위한 새로운 IRNet(Instance Relation Network)을 제안한다. 이는 각 세포 인스턴스 특징 간에 정보를 전달하기 위한 세포 연관성 행렬(cell association matrix)을 구성하는 Instance Relation Module (IRM)을 통해 이루어진다. 또한, 분류와 위치 정확도 간의 불일치를 해소하기 위해 희소성 제약이 있는 Duplicate Removal Module (DRM)을 제안한다.

## 2. 핵심 아이디어

IRNet의 핵심 아이디어는 중첩된 자궁경부 세포 분할을 위해 인스턴스 관계 상호작용을 활용하여 각 인스턴스의 특징 표현을 개선하는 것이다. 세포 덩어리 내에서 개별 세포의 외관이 강한 유사성을 공유한다는 임상적 관찰에 착안하여, 다른 인스턴스에서 얻은 상황 정보를 활용하여 의미론적 일관성을 높이고 모호한 경계 예측을 완화한다. 특히, Instance Relation Module (IRM)은 인스턴스 간의 협력적 상호작용을 활용하여 특징을 개선하며, Duplicate Removal Module (DRM)은 희소성 제약을 통해 분류 및 위치 정확도 간의 불일치를 보정하여 제안(proposal) 선택에 도움을 준다.

## 3. 상세 방법 설명

제안된 IRNet은 Fig. 1에 나타난 바와 같이 2단계 제안 기반 인스턴스 분할 패러다임(two-stage proposal based instance segmentation paradigm)을 따른다.
입력 이미지는 먼저 RPN(Region Proposal Network)에 공급되어 객체 후보를 생성한다. 그 다음 후보 특징은 RoIAlign 레이어에 의해 추출되어 검출 및 분할을 위한 두 가지 브랜치를 통과한다. 세포 덩어리에서 후보 선택의 네트워크 능력을 강화하고 상자 내 특징(in-box features) 간의 의미론적 일관성을 개선하기 위해, 검출 및 분할 헤드 뒤에 Duplicate Removal Module (DRM) 및 Instance Relation Module (IRM)을 추가하여 다른 세포 간의 상황 정보를 활용한다.

### Instance Relation Module (IRM)

IRM은 각 인스턴스의 임베디드 특징을 받아 인스턴스 연관성 행렬(instance association matrix)을 계산한 다음 인스턴스 관계에 따라 특징 간의 메시지를 파싱한다. 구체적으로, 인코더는 예측된 마스크와 딥 특징의 조합을 입력으로 받아 각 후보에 대한 융합 특징(fused features)을 생성한다. $n$을 이미지 내 인스턴스 수라고 할 때, 자기 주의 메커니즘(self-attention mechanism)이 $n$개 인스턴스 간의 연관성을 구축하는 데 사용된다.

IRM은 먼저 융합 특징을 채널별로 집계하여 16개의 특징 $B_j$($j=1,2 \dots 16$)를 구성한다. 각 $B_j$는 $\mathbb{R}^{c \times hw}$로 재형성되고, 전치 행렬과 곱해져 채널별 인스턴스 연관성 $D_j = B_j B_j^T$를 계산한다. 전체 인스턴스 연관성 행렬은 모든 채널별 연관성 행렬을 평균한 후 Softmax 레이어를 통해 정규화되어 $D = Softmax(avg(D_1,D_2 \dots D_c))$로 얻어진다. 여기서 $d_{pq}$는 $D$의 $p$번째 행, $q$번째 열 항목을 나타낸다. $A_p$와 $A_q$가 $p$번째 및 $q$번째 인스턴스 특징을 나타낼 때, 관계 상호작용 후 증강된 특징 $A_p'$는 다음과 같이 얻어진다:
$$ A*p' = \gamma \sum_{q=1}^n w_{pq} A_q + A_p $$
여기서 $\gamma$는 학습 가능한 스칼라 계수이다. 모든 인스턴스와 연관함으로써 증강된 특징은 다른 인스턴스 영역의 상황 정보로부터 이점을 얻어 의미론적 일관성을 높인다. 그런 다음 2배 스트라이드(stride)를 가진 하나의 $2 \times 2$ 역컨볼루션(deconvolution) 레이어를 통과한 후 분류자 역할을 하는 컨볼루션 레이어를 통과하여 예측된 마스크를 출력한다. 훈련 중에는 ground truth 마스크에 대해 이진 교차 엔트로피(Binary Cross-Entropy, BCE) 손실($L_{IRM}$)이 계산된다.

### Duplicate Removal Module (DRM)

객체성 점수(objectness score)를 직접적으로 NMS(Non-Maximum Suppression)에 활용하는 것은 분류 및 위치 정확도 간의 불일치로 인해 최적이 아닌 결과를 초래하며, 이는 세포 덩어리의 제안에서 더욱 심각하다. 이 불일치를 보정하기 위해 [5]에서 제안된 Duplicate Removal Module (DRM)은 외관 및 위치 특징을 입력으로 받아 관계 상호작용 후 변환된 특징을 사용하여 BCE 손실($L_{DRM}$)을 통해 제안이 올바른지 또는 중복인지 직접 예측한다. Pap smear 이미지의 세포는 하나의 큰 덩어리 대신 여러 로컬 소규모 클러스터에 모여 있다는 관찰을 기반으로, 모듈이 제안의 부분 집합 간의 상호작용에 집중할 수 있도록 DRM에 희소성 제약(sparsity constraint)을 추가할 것을 제안한다. 구체적으로, 각 대상에 대해 메시지 파싱을 위해 상위 $k$개($k=40$) 관계 가중치를 가진 제안만 사용한다. 또한, 중복 제거를 위해 예측된 확률을 직접 사용하는 대신, 분류 점수와 예측된 확률의 곱을 NMS에 사용하여 중첩 비율에 대한 강력한 제약을 부여한다.

### 전체 손실 함수 및 최적화

다중 작업 손실은 $L = L_{cls} + L_{reg} + L_{seg} + \alpha L_{DRM} + \beta L_{IRM}$로 정의된다. 여기서 $L_{DRM}$은 DRM에서 올바르거나 중복된 분류를 위한 BCE 손실이며, 올바른 것은 해당 ground truth에 대해 최대 IoU(Intersection over Union)를 가진 예측 바운딩 박스로 정의되고, 다른 중복 박스는 중복으로 정의된다. $L_{IRM}$은 IRM 후 정제된 마스크에 대한 픽셀별 BCE 손실이다. $\alpha$와 $\beta$는 손실 가중치 균형을 위한 하이퍼파라미터이다.

## 4. 실험 및 결과

### 데이터셋 및 평가 지표

액상 기반 Pap test 표본 82개 환자로부터 수집하여 CPS 데이터셋을 구축했다. 표본은 ×40 배율로 이미징되어 이미지 해상도는 픽셀당 약 0.2529 μm이다. 이후 1000×1000 해상도의 413개 이미지로 잘라냈다. 총 4439개의 세포질과 4789개의 핵이 세포병리학자에 의해 주석 처리되었다. 데이터셋은 환자 수준에서 7:1:2 비율로 훈련, 검증 및 테스트 세트로 분할되었다. 정량적 평가를 위해 픽셀 및 객체 수준을 모두 고려하는 Average Jaccard Index (AJI) [7]가 사용되었다. F1-score (F1)는 [2]를 참조하여 검출 정확도를 측정하는 데 사용되었다.

### 구현 세부 정보

IRNet은 PyTorch 1.0으로 구현되었다. 네트워크 아키텍처는 ResNet-50-FPN(Feature Pyramid Network with 50-layer ResNet) 조건에서 [8]과 동일하다. NVIDIA TITAN Xp 그래픽 카드와 CUDA 9.0 및 cuDNN 6.0이 계산에 사용되었다. 훈련 중에는 0.9 모멘텀을 가진 SGD가 최적화 도구로 사용되었다. 초기 학습률은 바이어스에 대해 2배의 계수를 가진 0.0025로 설정되었고, 가중치 감소(weight decay)는 0.0001로 설정되었다. 첫 500번의 반복에서는 학습률을 1/3의 warm-up 계수로 선형적으로 warm-up했다.

### 제안된 IRNet의 효과

중첩 세포 분할을 위한 다양한 알고리즘을 비교하는 실험을 수행했다. 비교 대상은 JOMLS [10], CSPNet [1], Mask R-CNN [4](DRM 및 IRM이 없는 IRNet), IRNet w/o IRM, IRNet w/o DRM, 그리고 제안된 IRNet이다.
모든 딥러닝 기반 방법은 레벨셋 기반 방법 [10]에 비해 더 나은 성능을 달성했다. 이는 CPS 데이터셋이 [10]에서 사용된 것보다 백혈구 및 기타 이물질을 포함하는 더 복잡한 배경 정보를 가지고 있기 때문이다.
기준 모델(Mask R-CNN)과 비교하여 DRM을 추가하면 (IRNet w/o IRM) 세포질과 핵의 F1 점수가 각각 9.03%와 3.23% 향상되어, 중복된 박스를 효과적으로 억제할 수 있음을 입증한다.
분할 브랜치 뒤에 IRM을 추가하면 (IRNet w/o DRM) 세포질과 핵의 AJI가 각각 2.19%와 5.84% 향상되어, 인스턴스 연관성을 활용하여 특징을 전달함으로써 증강된 특징이 의미론적 표현에 대해 더 일관되고 판별적임을 보여준다.
DRM과 IRM을 결합한 제안된 IRNet은 세포질과 핵의 AJI에서 각각 4.97%와 6.33%, F1 점수에서 12.5%와 3.62%의 향상을 보여 다른 방법들을 크게 능가한다.

### Instance Relation Module (IRM)에 대한 Ablation Study

제안된 IRM의 설계에 대한 Ablation Study를 수행했다.

1. **DF (Deep Feature)**: 분할 브랜치의 역컨볼루션 레이어 전 딥 특징을 사용.
2. **MSK (Mask)**: 분할 브랜치에서 예측된 마스크를 사용.
3. **RL (Relation Learning)**: 채널별 인스턴스 관계 학습 수행.
    관계 학습을 포함한 모든 방법이 더 나은 성능을 보인다.
    IRM이 없는 IRNet과 비교하여 딥 특징을 직접 활용한 관계 상호작용(DF+RL)은 세포질과 핵의 AJI에서 각각 3.05%와 0.47% 향상을 가져온다.
    예측된 마스크를 직접 사용하는 경우(MSK+RL)는 세포질의 AJI에서 IRNet w/o IRM보다 2.67% 우수하다.
    딥 특징과 선택된 마스크를 동시에 사용하면 (DF+RL)보다 세포질과 핵에서 각각 4.33%와 2.88%의 성능 향상을 보여 IRM 설계의 효과를 입증한다.

### 정성적 비교

Fig. 3은 세포질의 심한 폐색 및 흩어져 있는 백혈구와 같은 어려운 경우를 포함하는 테스트 세트의 대표적인 샘플을 보여준다. 기존 방법(JOMLS)은 이물질과 핵을 식별하는 데 실패한다. Mask R-CNN과 비교하여 IRM을 추가하면 세포 덩어리의 모호한 세포질 경계 예측을 완화한다. 또한, DRM을 결합하면 중복 예측을 성공적으로 억제한다.

## 5. 강점, 한계

**강점**: 제안된 IRNet은 인스턴스 관계 정보를 활용하여 더 나은 의미론적으로 일관된 특징 표현을 제공하며, 결과적으로 더 일관된 경계 모양의 마스크를 생성한다. 정량적 및 정성적 결과는 그 효과를 입증한다. 제안된 IRM은 본질적으로 일반적이며 다양한 제안 기반 인스턴스 분할 방법과 상호 보완적으로 사용될 수 있다.

**한계**: 논문에서 명시적인 한계는 언급되지 않았다.

## 6. 결론

Pap smear 이미지에서 세포질 및 핵 인스턴스를 정확하게 분할하는 것은 추가적인 세포 형태 분석 및 자궁경부암 등급 분류에 매우 중요하다. 본 논문에서는 인스턴스 관계 정보를 활용하여 더 나은 의미론적으로 일관된 특징 표현을 제공하는 새로운 IRNet을 제안했다. 다른 인스턴스의 특징을 집계함으로써, 이들은 더 일관된 경계 모양의 마스크를 생성하는 경향이 있다. 정량적 및 정성적 결과는 제안된 방법의 효과를 입증한다. 제안된 IRM은 본질적으로 일반적이며 다양한 제안 기반 인스턴스 분할 방법과 상호 보완적으로 사용될 수 있다.
