# Proposal-free Network for Instance-level Object Segmentation

- **저자**: Xiaodan Liang, Yunchao Wei, Xiaohui Shen, Jianchao Yang, Liang Lin, Shuicheng Yan
- **발표연도**: 2015
- **arXiv**: https://arxiv.org/abs/1509.02636

## 1. 논문 개요

본 논문은 객체 수준(instance-level) 객체 분할이라는 중요한 반면 아직 충분히 탐구되지 않은 과제를 해결하기 위해 제안-자유 네트워크(Proposal-Free Network, PFN)를 제안한다. 기존의 접근 방식들이 대부분 영역 제안(region proposal) 방법을 기반으로 후보 영역을 추출하고 객체 분류를 통해 최종 결과를 도출하는 반면, 이러한 정확한 영역 제안을 생성하는 과정 자체가 매우 어렵다는 문제점이 있었다. PFN은 이러한 어려움을 해결하기 위해 픽셀 단위의 딥 컨볼루션 신경망을 기반으로 다양한 카테고리의 인스턴스 개수와 각 픽셀이 속하는 인스턴스 바운딩 박스의 좌표, 그리고 각 픽셀에 대한 다양한 카테고리의 신뢰도를 출력한다. 이 모든 출력은 간단한 후처리 단계에서 기성 클러스터링 방법을 사용하여 최종 객체 수준 객체 분할 결과를 자연스럽게 생성할 수 있다. PFN 전체는 제안 생성 단계 없이 종단 간(end-to-end) 방식으로 쉽게 훈련될 수 있다.

## 2. 핵심 아이디어

PFN의 중심적인 아이디어는 기존의 객체 수준 객체 분할 방법론들이 필수적으로 포함했던 복잡한 영역 제안 생성 단계를 완전히 제거하는 것이다. 인간이 이미지를 한 번 훑어보고 이미지 내의 객체가 무엇이고 어디에 있는지 즉시 아는 방식에서 영감을 받아, PFN은 전역 이미지 맥락에서 객체 인스턴스의 영역을 직접 추론하여 인스턴스 수준 분할 작업을 재구성한다. 이 접근 방식은 픽셀들이 유사한 인스턴스 위치를 예측하는 경우 직접적으로 동일한 객체 인스턴스 영역으로 클러스터링될 수 있다는 직관에 기반한다. 가려진(occluded) 객체의 경계는 예측된 인스턴스 위치의 차이를 통해 추론될 수 있다. 이를 위해 PFN은 다음 세 가지 하위 작업을 동시에 처리한다: 카테고리 수준 분할, 각 픽셀에 대한 인스턴스 위치 예측, 전체 이미지에서 각 카테고리에 대한 인스턴스 개수 예측.

## 3. 상세 방법 설명

PFN의 상세 네트워크 아키텍처는 세 가지 목표(카테고리 수준 분할, 각 픽셀에 대한 인스턴스 위치, 모든 카테고리에 대한 인스턴스 개수)를 동시에 학습한다.

### 3.1. 카테고리 수준 분할 예측

PFN은 밀집된 카테고리 수준 분할 작업을 위해 공개적으로 사용 가능한 사전 훈련된 VGG 16-레이어 분류 네트워크를 기반으로 미세 조정된다. 특히, 선도적인 정확도와 효율성 때문에 "DeepLab-CRF-LargeFOV" 네트워크 구조를 기본으로 활용한다. VGG-16의 마지막 레이어에서 1000가지 ImageNet 분류기는 $C+1$ 신뢰도 맵으로 대체되며, 여기서 $C$는 카테고리 개수이다. 손실 함수는 신뢰도 맵(원본 이미지에 비해 8배 다운샘플링됨)의 픽셀 단위 교차 엔트로피(cross-entropy) 합이다. 테스트 시에는 완전히 연결된 조건부 확률 필드(fully-connected conditional random fields)를 사용하여 더 부드럽고 정확한 분할 맵을 생성한다.

이 카테고리 수준 네트워크는 후속 인스턴스 수준 분할을 위한 시맨틱 분할 마스크를 생성한다. 그 다음 인스턴스 수준 네트워크는 카테고리 수준 네트워크를 기반으로 미세 조정된다. 두 네트워크를 별도의 단계로 최적화하는 이유는 카테고리 수준 분할은 특정 카테고리의 다른 객체 인스턴스에 둔감한 예측을 선호하는 반면, 인스턴스 수준 분할은 개별 인스턴스를 구별하는 것을 목표로 하기 때문이다. 이전에 훈련된 카테고리 수준 네트워크의 컨볼루션 필터를 활용하여 인스턴스 수준 네트워크의 파라미터를 초기화하며, 이는 더 나은 파라미터 학습 시작점을 제공한다.

### 3.2. 인스턴스 수준 분할 예측

인스턴스 수준 분할 네트워크는 임의 크기의 이미지를 입력으로 받아 모든 픽셀에 대한 해당 인스턴스 위치와 각 카테고리의 인스턴스 개수를 출력한다.

- **픽셀 단위 인스턴스 위치 예측**: 각 이미지에 대해 각 픽셀의 인스턴스 위치 벡터는 픽셀을 포함하는 해당 객체 인스턴스의 바운딩 박스 정보로 정의된다. PFN은 각 인스턴스 바운딩 박스의 중심($c^x, c^y$), 좌상단 모서리($x_1, y_1$), 우하단 모서리($x_2, y_2$)의 좌표를 포함하는 6차원 위치 맵을 예측한다. 이 예측된 좌표는 서로 보완적이며 가깝거나 가려진 인스턴스를 처리하는 알고리즘을 더욱 견고하게 만든다. 더 정확한 픽셀 단위 인스턴스 위치 예측을 위해, 초기 미세 레이어의 지역적 세부 정보와 깊고 거친 레이어의 전역 시맨틱 정보를 공동으로 인코딩하기 위해 개별 감독(즉, 다중 손실)을 갖춘 다중 스케일 예측 스트림이 추가된다. 최종 예측 레이어 전에 다중 스케일 예측을 결합하는 융합(fusion) 레이어가 활용된다.
  손실 함수는 다음과 같이 정의된다:
  $$ L^o(\mathbf{t},\mathbf{t^{_}})=\sum*{m=1}^{M}\frac{\sum*{i}\ell^{o}\_{m}(t*{i},t\_{i}^{_})}{\Omega}+\frac{\sum*{i}\ell^{o}\_{fuse}(t*{i},t*{i}^{_})}{\Omega} $$
    여기서 $\mathbf{t}$와 $\mathbf{t^{_}}$는 각각 모든 픽셀의 예측된 인스턴스 위치와 실제 인스턴스 위치이며, $\Omega$는 실제 분할 맵의 총 픽셀 수이다.

- **인스턴스 개수 예측**: PFN의 또 다른 하위 작업은 모든 카테고리의 인스턴스 개수를 예측하는 것이다. 입력 이미지의 인스턴스 개수 벡터는 카테고리 수준 정보(특정 카테고리의 인스턴스 출현 여부)와 인스턴스 수준 정보(특정 카테고리에 대한 객체 인스턴스 개수)를 포함한다. 카테고리 수준 분할 네트워크의 마지막 컨볼루션 레이어의 특징 맵과 융합 레이어 이후의 인스턴스 수준 특징 맵이 함께 연결되어 융합된 특징 맵을 형성하며, 이는 인스턴스 개수를 공동으로 예측하는 데 활용된다. 인스턴스 개수 예측을 위한 유클리드 손실은 다음과 같다:
  $$ L^n(\mathbf{g},\mathbf{g^{_}})=\frac{1}{C}\sum*{c=1}^{C}||g*{c}-g^{_}_{c}||^{2} $$
  여기서 $\mathbf{g}=[g_{1},g*{2},...,g*{C}]$는 예측된 인스턴스 개수 벡터, $\mathbf{g^{*}}=[g^{*}_{1},g^{*}_{2},...,g^{*}_{C}]$는 실제 인스턴스 개수 벡터이다.

- **네트워크 훈련**: 전체 인스턴스 수준 네트워크를 훈련하기 위한 전체 손실 함수 $L$은 다음과 같이 표현된다:
  $$ L(\mathbf{t},\mathbf{t^{_}},\mathbf{g},\mathbf{g^{_}})=\lambda L^{o}(\mathbf{t},\mathbf{t^{_}})+L^{n}(\mathbf{g},\mathbf{g^{_}}) $$
    클래스 균형 매개변수 $\lambda$는 경험적으로 10으로 설정되어 픽셀 단위 인스턴스 위치 예측에 대한 편향을 의미한다. 이를 통해 인스턴스 개수 예측과 픽셀 단위 인스턴스 위치 예측이 통합된 네트워크에서 공동으로 최적화된다.

### 3.3. 테스트 단계

테스트 단계에서는 먼저 입력 이미지를 카테고리 수준 분할 네트워크에 입력하여 카테고리 수준 분할 결과를 얻은 다음, 인스턴스 수준 네트워크에 입력하여 인스턴스 개수 벡터 $\mathbf{g}$와 픽셀 단위 인스턴스 위치 예측 $\mathbf{t}$를 얻는다. 그런 다음 모든 픽셀의 예측된 인스턴스 위치 $\mathbf{t}$를 기반으로 클러스터링을 수행한다. 각 카테고리에 대해 예측된 인스턴스 위치를 별도로 클러스터링하며, 이는 카테고리 수준 분할 결과 $\mathbf{p}$로 $\mathbf{t}$를 필터링하여 얻을 수 있다. 모든 카테고리의 예측된 인스턴스 개수 $\mathbf{g}$는 각 카테고리에 대한 클러스터 개수를 나타내는 데 사용된다. 클러스터링을 위해 각 카테고리에 대한 쌍별 유사도 행렬(pairwise affinity matrix)이 계산된다. 두 픽셀 $i$와 $j$ 사이의 유사도 $w_{i,j}$는 다음과 같이 정의된다:
$$ w*{i,j}=\exp(\frac{-||t*{i}-t*{j}||^{2}/|t*{i}|}{2\sigma^{2}})+\exp(\frac{-||q*{i}-q*{j}||^{2}/|q\_{i}|}{2\sigma^{2}}) $$
여기서 $|t_i|$는 벡터 $t_i$의 특징 차원, $|q_i|$는 공간 좌표 $q_i$의 특징 차원이다. $\sigma$는 모든 카테고리에 대해 20으로 설정된다. 예측된 전역 이미지 카테고리와 픽셀 단위 카테고리 수준 분할 간의 불일치가 종종 관찰되므로, 인스턴스 개수 예측을 사용하여 생성된 카테고리 수준 분할을 개선한다. 또한, 노이즈가 있는 배경 픽셀로 인해 예측된 분할 결과가 완벽하지 않으므로, 픽셀 수가 분할 마스크 픽셀의 0.1% 미만인 클러스터는 실험적으로 제거한다.

## 4. 실험 및 결과

### 4.1. 실험 설정

PFN은 PASCAL VOC 2012 유효성 검사 분할 벤치마크 [4]에서 광범위하게 평가되었다. SDS [9] 및 Chen 외 [16]의 방법과 비교되었다. SBD [8]의 분할 주석은 네트워크 훈련에 사용되었고, PASCAL VOC 2012 분할 유효성 검사 세트의 1,449개 이미지가 평가에 사용되었다. 모든 네트워크는 공개된 DeepLab 코드 [20]를 기반으로 훈련 및 테스트되었으며, Caffe 플랫폼 [10]에서 단일 NVIDIA GeForce Titan GPU(6GB 메모리)를 사용하여 구현되었다. 두 훈련 단계 모두에서 모든 새 레이어는 표준 편차 0.01의 제로 평균 가우시안 분포에서 가중치를 무작위로 초기화했다. 미니 배치 크기는 8개 이미지, 사전 훈련된 레이어의 초기 학습률은 0.001, 새로 추가된 레이어는 0.01이었다. 20 에포크 후에 학습률을 1/10로 줄이고 두 네트워크를 약 60 에포크 동안 훈련했다. 모멘텀과 가중치 감쇠는 각각 0.9와 0.0005로 설정되었다. PFN은 300x500 이미지를 약 1초 만에 처리할 수 있으며, 이는 이전 최신 기술보다 훨씬 빠르다 (SDS는 약 40초 소요).

### 4.2. 결과 및 비교

첫 번째 훈련 단계에서 "DeepLab-CRF-LargeFOV" [20]와 동일한 아키텍처를 사용하여 카테고리 수준 분할 네트워크를 훈련했다. 21개 클래스에 걸쳐 평균 픽셀 교차-over-union (IOU) [17] 측면에서 픽셀 단위 분할을 평가하여 PASCAL VOC 2012 유효성 검사 세트 [4]에서 카테고리 수준 분할 작업에서 67.53%를 달성했으며, 이는 [20]에서 보고된 67.64%보다 약간 낮은 수준이다. PFN은 PASCAL VOC 2012 유효성 검사 세트에서 0.5 IoU에서 20개 클래스에 대한 $AP^r$이 58.7%에 도달하여, 최신 기술인 SDS [9]의 43.8% 및 [16]의 46.3%보다 훨씬 높다. 이는 PFN이 객체 수준 객체 분할에서 최신 기술을 크게 능가함을 보여준다.

### 4.3. 네트워크의 Ablation Studies

PFN의 6가지 중요 구성 요소(훈련 전략, 인스턴스 위치 예측, 네트워크 구조, 인스턴스 개수 예측, 테스트 전략 및 상한)의 효과를 평가했다.

- **훈련 전략**: PFN의 두 단계 훈련(카테고리 수준 및 인스턴스 수준 네트워크)의 필요성을 입증하기 위해, 세 가지 하위 작업을 하나의 학습 단계에서 최적화하는 통합 네트워크("PFN unified")의 성능을 평가했다. "PFN unified"는 평균 $AP^r$에서 9.6% 감소를 보였다.
- **인스턴스 위치 예측**: PFN은 각 픽셀에 대해 바운딩 박스 중심, 좌상단, 우하단의 6차원 공간 좌표를 예측한다. 다른 5가지 종류의 인스턴스 위치 예측도 평가했으며, "PFN offsets (2)"는 "PFN"보다 현저히 낮은 성능을 보였다 ($AP^r$에서 51.0% vs 58.7%).
- **네트워크 구조**: 다중 스케일 예측의 효과가 검증되었다. "PFN w/o multiscale"은 픽셀 단위 인스턴스 위치를 예측하기 위해 하나의 직접적인 레이어만 사용하는 성능을 보여주며, $AP^r$에서 3.5% 감소했다. 공간 좌표를 특징 맵으로 사용하는 것의 우수성도 "PFN w/o coordinate maps"와 "PFN"을 비교하여 입증되었다 (0.7% $AP^r$ 차이).
- **인스턴스 개수 예측**: 카테고리 수준 정보 없이 인스턴스 위치 예측만을 특징 맵으로 사용하여 인스턴스 개수를 예측하는 "PFN w/o category-level"은 "PFN" (51.4% vs 58.7%)과의 큰 격차를 통해 카테고리 수준 정보의 중요성을 확인했다.
- **테스트 전략**: 스펙트럼 클러스터링 시 두 픽셀의 유사도가 6차원 예측 인스턴스 위치와 각 픽셀의 두 공간 좌표를 모두 고려하여 계산된다. 유사도 함수에서 좌표를 제거하면 "PFN w/o coordinates"와 "PFN"을 비교하여 $AP^r$에서 55.3% vs 58.7%로 상당한 감소가 관찰되었다.
- **상한**: "PFN upperbound_instnum"은 클러스터링을 위해 실제 인스턴스 개수 예측을 사용하는 성능을 보고하며, $AP^r$에서 2.6% 개선만 보였다.

### 4.4. 시각적 설명

PFN을 사용한 예측된 인스턴스 수준 분할 결과는 시각적으로 설명되었다. PFN은 객체 수준 분할 결과를 직접 생성하며, 이는 기존 방법들이 각 인스턴스에 대해 여러 영역 제안을 생성하고 상위 탐지 결과만 시각화하는 방식과 대비된다. PFN은 심하게 가려진 객체, 복잡한 배경 혼란, 다양한 외관 패턴을 가진 객체 인스턴스를 예측하는 데 좋은 성능을 보였다.

## 5. 강점, 한계

### 강점

- 객체 수준 객체 분할을 위한 제안-자유(Proposal-free) 접근 방식을 사용하여, 어려운 영역 제안 생성 단계를 제거한다.
- 인스턴스 개수, 픽셀 단위 바운딩 박스 좌표, 카테고리 신뢰도를 출력한다.
- 간단한 후처리(simple post-processing)를 위해 모든 기성 클러스터링 방법을 사용할 수 있다.
- 종단 간(end-to-end) 방식으로 쉽게 훈련될 수 있다.
- PASCAL VOC 2012 벤치마크에서 최신 기술을 크게 능가한다 ($AP^r$ 성능을 46.3%에서 58.7%로 향상).
- 이전 방법론에 비해 구현이 더 간단하고 계산 비용이 훨씬 낮다 (300x500 이미지를 약 1초 만에 처리).
- 심하게 가려진 객체, 복잡한 배경 혼란, 복잡한 장면, 다양한 외관 패턴 및 작은 객체 인스턴스를 예측하는 데 탁월한 성능을 보인다.

### 한계

- 매우 심하게 가려진 객체 인스턴스를 분할하지 못할 수 있다.
- 일부 작은 객체 인스턴스를 분할하지 못할 수 있다.
- 향후 연구에서 실외 및 실내 장면의 일반적인 다중 인스턴스(더 높은 수준의 혼란 및 가려짐이 있을 수 있음)로 프레임워크를 확장해야 한다 (이는 현재 이러한 복잡한 시나리오에서의 한계를 시사).

## 6. 결론

본 논문은 세분화된 객체 수준 분할을 위한 효과적인 제안-자유 네트워크(PFN)를 제시한다. PFN은 추가적인 영역 제안 방법을 필수적으로 사용하는 대신, 특정 인스턴스에 속하는 각 픽셀의 인스턴스 위치 벡터와 모든 카테고리의 인스턴스 개수를 직접 예측한다. 동일하거나 유사한 인스턴스 위치를 예측하는 픽셀은 직접적으로 동일한 객체 인스턴스에 속하는 것으로 간주될 수 있다. 테스트 시에는 예측된 픽셀 단위 인스턴스 위치에 대해 간단한 스펙트럼 클러스터링이 수행되어 최종 인스턴스 수준 분할 결과를 생성하며, 모든 카테고리의 예측된 인스턴스 개수는 각 카테고리에 대한 클러스터 개수를 나타내는 데 사용된다. PFN은 PASCAL VOC 2012 분할 벤치마크에서 최신 기술에 비해 상당한 개선을 달성했다. PFN의 다양한 구성 요소에 대한 광범위한 평가는 제안된 방법의 효과를 검증한다. PFN은 복잡한 전처리 및 후처리 과정 없이 이전 최신 기술에 비해 구현이 훨씬 간단하고 계산 비용이 훨씬 낮다. 향후에는 프레임워크를 실외 및 실내 장면의 일반적인 다중 인스턴스로 확장할 계획이며, 이는 더 높은 수준의 혼란과 가려짐을 포함할 수 있다.
