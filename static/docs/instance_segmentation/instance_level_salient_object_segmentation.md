# Instance-Level Salient Object Segmentation

- **저자**: Guanbin Li, Yuan Xie, Liang Lin, Yizhou Yu
- **발표연도**: 2017
- **arXiv**: https://ar5iv.labs.arxiv.org/html/1704.03604v1

## 1. 논문 개요

이 논문은 이미지 내에서 객체 인스턴스를 식별할 수 없는 기존의 Salient Object Detection(현저한 객체 감지) 방법의 한계를 해결하기 위해 Salient Instance Segmentation(현저한 인스턴스 분할)이라는 새로운 방법을 제시한다. 제안된 방법은 입력 이미지에 대해 개별 객체 인스턴스 레이블이 있는 현저성 마스크를 생성한다. 이 방법은 현저성 맵 추정, 현저한 객체 윤곽선 감지, 현저한 객체 인스턴스 식별의 세 단계로 구성된다. 처음 두 단계를 위해 다중 스케일 현저성 개선 네트워크(Multiscale Saliency Refinement Network)를 제안하여 고품질의 현저한 영역 마스크와 현저한 객체 윤곽선을 생성한다. 이 네트워크는 다중 스케일 조합 그룹화(Multiscale Combinatorial Grouping) 및 MAP 기반 부분 집합 최적화 프레임워크와 통합되어 매우 유망한 현저한 객체 인스턴스 분할 결과를 생성한다. 또한, 현저한 인스턴스 분할의 추가 연구 및 평가를 위해 1000개의 이미지와 픽셀 단위의 현저한 인스턴스 주석으로 구성된 새로운 데이터베이스를 구축하였다. 실험 결과는 제안된 방법이 현저한 영역 감지를 위한 모든 공개 벤치마크 및 새로운 현저한 인스턴스 분할 데이터셋에서 최첨단 성능을 달성할 수 있음을 보여준다.

## 2. 핵심 아이디어

이 논문의 핵심 아이디어는 기존의 현저한 객체 감지(Salient Object Detection) 방법이 이미지에서 현저한 "영역"만 감지하고 개별 "객체 인스턴스"는 식별하지 못한다는 문제점을 해결하는 데 있다. 저자들은 이보다 더 도전적인 과제인 "Instance-Level Salient Object Segmentation" 또는 줄여서 "Salient Instance Segmentation"을 제안한다. 이는 감지된 현저한 영역 내에서 개별 객체 인스턴스를 식별하는 것을 목표로 한다.

이러한 목표를 달성하기 위해 Salient Instance Segmentation 작업은 다음 세 가지 하위 작업으로 분해된다.

1. **이진 현저성 맵 추정**: 입력 이미지에서 현저한 영역을 나타내는 픽셀 수준의 현저성 마스크를 예측한다.
2. **현저한 객체 윤곽선 감지**: 개별 현저한 객체 인스턴스에 대한 윤곽선 감지를 수행하여 객체 윤곽선 이외의 불필요한 경계를 억제하고 현저한 객체 제안(proposals) 생성을 유도한다.
3. **현저한 객체 인스턴스 식별**: 현저한 객체 제안을 생성하고, 현저한 영역을 가장 잘 커버하도록 작은 부분 집합의 제안을 선택한다. 마지막으로, CRF(Conditional Random Field) 기반의 정제 방법을 적용하여 현저한 객체 인스턴스의 공간적 일관성을 향상시킨다.

이러한 하위 작업들을 해결하기 위해, 논문은 다중 스케일 현저성 개선 네트워크(Multiscale Saliency Refinement Network, MSRNet)를 제안한다. MSRNet은 동일한 입력 이미지의 스케일링된 버전을 처리하는 세 개의 병렬 스트림과 각 스트림에서 다른 스케일의 결과를 융합하는 학습된 어텐션 모델로 구성된다. 이 네트워크는 기존 VGG 네트워크의 하향식(bottom-up) 및 상향식(top-down) 정보를 통합하도록 설계된 정제된 VGG 네트워크 아키텍처를 공유한다. 이렇게 감지된 현저한 객체 인스턴스의 윤곽선을 기반으로 다중 스케일 조합 그룹화(Multiscale Combinatorial Grouping, MCG)를 적용하여 현저한 객체 제안을 생성하고, MAP 기반 부분 집합 최적화를 통해 노이즈가 많거나 겹치는 제안을 필터링한다. 최종적으로, 완전 연결 CRF 모델을 사용하여 초기 현저한 인스턴스 분할의 공간적 일관성과 윤곽선 지역화를 개선한다.

## 3. 상세 방법 설명

제안된 현저한 인스턴스 분할 방법은 현저한 영역 감지, 현저한 객체 윤곽선 감지, 현저한 인스턴스 생성 및 현저한 인스턴스 정제라는 네 가지 계단식 구성 요소로 이루어져 있다.

### 3.1. 다중 스케일 정제 네트워크 (Multiscale Refinement Network, MSRNet)

MSRNet은 현저한 영역 감지 및 현저한 객체 윤곽선 감지를 이진 픽셀 레이블링 문제로 공식화한다. 현저한 객체는 다양한 스케일을 가질 수 있으므로, MSRNet은 세 개의 정제된 VGG 네트워크 스트림과 학습된 어텐션 모델을 사용하여 다른 스케일의 결과를 융합한다.

#### 3.1.1. 정제된 VGG 네트워크

- **구조**: 원본 VGG16을 완전 컨볼루션 네트워크로 변환하여 하향식 백본 네트워크로 사용한다. VGG16의 두 개의 완전 연결 계층은 $1 \times 1$ 컨볼루션 계층으로 변환된다. 마지막 두 풀링 계층의 서브샘플링을 건너뛰고, 준-최종(penultimate) 풀링 계층 이후의 컨볼루션 계층을 atrous convolution으로 대체하여 필터의 원래 수용 필드를 유지한다. 이로 인해 변환된 VGG 네트워크의 출력 해상도는 원본 입력 해상도의 $1/8$이 된다.
- **상향식 정제**: VGG16의 다섯 개 맥스 풀링 계층 각각에 추가 컨볼루션 계층( $3 \times 3$ 커널, 64 채널)을 연결하여 차원을 축소한다. "정제 모듈"($R$)을 통합하여 각 풀링 계층의 효과를 반전시키고 해상도를 두 배로 높인다. 이는 하위 계층의 고해상도 정보와 상위 계층의 의미론적 정보를 통합하여 정밀한 영역 및 윤곽선 감지 결과를 생성하는 데 필수적이다.

#### 3.1.2. 어텐션 가중치를 사용한 다중 스케일 융합

- **다중 스케일 처리**: 이전 섹션의 정제된 VGG 네트워크를 세 번 복제하여 입력 이미지의 세 가지 다른 스케일($s \in \{1, 0.75, 0.5\}$)을 처리한다.
- **픽셀 단위 융합**: MSRNet의 최종 출력은 세 가지 확률 맵의 픽셀 단위 가중치 합으로 계산되며, 각 픽셀에 대한 확률 점수의 가중치는 고정되지 않고 공간적으로 다양하다.
  $$ F_{c}=\Sigma_{s \in \{1,0.75,0.5\}} W^{s} \odot M\_{c}^{s}. $$
    여기서 $M_c^s$는 스케일 $s$에서 클래스 $c$에 대한 확률 맵이고, $W^s$는 어텐션 가중치(attentional weights)이다. 이 가중치는 다른 공간적 위치와 이미지 스케일의 특징에 얼마나 많은 주의를 기울여야 하는지를 반영한다.
- **어텐션 모듈**: 세 개의 상향식 정제 스트림에 있는 준-최종 계층의 출력 특징 맵을 연결한 것을 입력으로 받아 어텐션 가중치를 추론하기 위한 두 개의 컨볼루션 계층으로 구성된 어텐션 모듈을 MSRNet에 추가하여 어텐션 가중치를 학습한다.

#### 3.1.3. 다중 스케일 정제 네트워크 훈련

- **두 가지 모델 훈련**: 현저한 영역 감지와 현저한 객체 윤곽선 감지라는 두 가지 하위 작업을 수행하기 위해 동일한 MSRNet 아키텍처를 기반으로 두 개의 심층 모델을 훈련한다. 이 하위 작업들은 별도의 훈련 세트를 갖는다.
- **전이 학습**: 현저한 윤곽선 감지 훈련 이미지 수가 적기 때문에, 실제로는 먼저 현저한 영역 감지를 위한 네트워크를 훈련하고, 이 훈련된 네트워크의 복사본을 현저한 윤곽선 감지를 위해 추가로 미세 조정한다.
- **손실 함수 가중치**: 두 하위 작업의 손실 함수는 샘플 균형을 위해 다른 가중치를 사용한다. 현저한 객체 윤곽선 감지의 경우 "윤곽선" 픽셀을 잘못 분류하는 것에 대한 페널티가 "비윤곽선" 픽셀에 대한 페널티의 10배이며, 현저한 영역 감지의 경우 "현저한" 픽셀을 잘못 분류하는 것에 대한 페널티가 "비현저한" 픽셀에 대한 페널티의 2배이다.
- **초기화 및 최적화**: 현저한 영역 감지를 위해 MSRNet을 훈련할 때, 하향식 백본 네트워크는 ImageNet에서 사전 훈련된 VGG16 네트워크로 초기화하고, 상향식 정제 스트림은 무작위 값으로 초기화한다. MSRNet의 세 개의 정제된 VGG 네트워크는 공동으로 미세 조정되며, 공유 매개변수는 표준 확률적 경사 하강법(Stochastic Gradient Descent, SGD)을 사용하여 최적화된다.

### 3.2. 현저한 인스턴스 제안 (Salient Instance Proposal)

- **MCG 기반 제안**: MSRNet 기반의 현저한 객체 윤곽선 감지 결과를 기반으로 다중 스케일 조합 그룹화(MCG) 알고리즘을 사용하여 현저한 객체 제안을 생성한다. MCG는 하향식 계층적 이미지 분할 및 객체 후보 생성을 위한 통합 접근 방식이다.
- **계층적 분할**: 입력 이미지로부터 네 개의 현저한 객체 윤곽선 맵(세 개는 스케일링된 입력에서, 하나는 융합된 맵에서)을 생성한다. 각 윤곽선 맵은 ultrametric contour map (UCM)으로 표현되는 고유한 계층적 이미지 분할을 생성하는 데 사용된다. 이 네 가지 계층은 정렬되어 단일 계층적 분할로 결합된다.
- **제안 필터링 및 최적화**: 높은 재현율을 보장하기 위해 이미지당 800개의 현저한 객체 제안을 생성한다. 이 중 $80\%$ 미만의 현저한 픽셀을 포함하는 제안은 버린다. 초기 선별된 현저한 객체 제안 집합에 대해 MAP 기반 부분 집합 최적화 방법을 추가로 적용하여 압축된 객체 제안 집합을 생성한다. 이 압축된 집합에 남아있는 객체 제안의 수가 이미지에서 예측된 최종 현저한 객체 인스턴스 수가 된다.

### 3.3. 현저한 인스턴스 분할 정제 (Refinement of Salient Instance Segmentation)

- **CRF 모델**: 현저한 객체 제안과 현저한 영역이 독립적으로 얻어지기 때문에, 감지된 모든 현저한 인스턴스의 합집합과 감지된 모든 현저한 영역의 합집합 사이에 불일치가 존재할 수 있다. 이를 해결하기 위해 완전 연결 CRF(Conditional Random Field) 모델을 제안하여 초기 현저한 인스턴스 분할 결과를 정제한다.
- **배경 처리**: $K$개의 현저한 인스턴스가 있다고 가정할 때, 배경은 $K+1^{th}$ 클래스로 간주된다.

## 4. 실험 및 결과

- **데이터셋**: 현저한 인스턴스 분할의 추가 연구 및 평가를 위해 1000개의 이미지와 픽셀 단위의 현저한 인스턴스 주석으로 구성된 새로운 데이터셋을 구축하였다.
- **벤치마크**: 현저한 영역 감지를 위한 모든 공개 벤치마크 및 새로 구축된 현저한 인스턴스 분할 데이터셋에서 실험을 수행했다.
- **정량적/정성적 결과**: 제안된 방법은 기존의 현저한 영역 감지 방법들에 비해 현저히 높은 정밀도(precision)를 달성하며, 모든 공개 벤치마크에서 최첨단 성능을 기록하였다. 특히, MSRNet은 현저한 윤곽선 감지에서도 잘 일반화되어 감지된 현저한 영역에서 뚜렷한 객체 인스턴스를 분리하는 데 성공적이다.
- **실제 결과**: 객체 제안 생성 및 선별 기술과 통합될 때, 본 방법은 고품질의 분할된 현저한 객체 인스턴스를 생성할 수 있음을 입증하였다. 새로 구축된 데이터셋에서도 우수한 성능을 보여주며, 현저한 인스턴스 분할이라는 새로운 과제에 대한 효과적인 해결책을 제시하였다.

## 5. 강점, 한계

### 강점 (Strengths)

- **새로운 문제 정의 및 해결**: 기존 현저한 객체 감지 방법의 한계인 개별 객체 인스턴스 식별 불가능 문제를 해결하기 위해 "Instance-Level Salient Object Segmentation"이라는 새로운 과제를 정의하고 효과적인 해결책을 제시하였다.
- **다중 스케일 처리 능력**: MSRNet은 입력 이미지의 여러 스케일을 병렬로 처리하고 어텐션 메커니즘을 통해 결과를 융합하여 다양한 크기의 현저한 객체를 효과적으로 처리할 수 있다.
- **통합된 네트워크 아키텍처**: 현저한 영역 감지와 현저한 객체 윤곽선 감지를 하나의 MSRNet 아키텍처 내에서 통합하여 수행함으로써 두 관련 작업의 시너지를 활용한다.
- **최첨단 성능**: 기존의 현저한 영역 감지 벤치마크에서 최첨단 성능을 달성했으며, 새로 구축된 현저한 인스턴스 분할 데이터셋에서도 유망한 결과를 보여주었다.
- **새로운 데이터셋 구축**: 현저한 인스턴스 분할이라는 새로운 연구 분야의 발전을 위해 1000개의 이미지와 픽셀 단위의 인스턴스 주석을 포함하는 도전적인 데이터셋을 구축한 것은 중요한 기여이다.

### 한계 (Limitations)

- **단계별 구성의 복잡성**: 현저한 영역 감지, 윤곽선 감지, 인스턴스 제안 및 정제 등 여러 계단식 구성 요소로 이루어져 있어 전체 시스템의 복잡도가 높고, 각 단계에서의 오류가 다음 단계로 전파될 가능성이 있다.
- **MAP 기반 최적화의 잠재적 문제**: 논문은 기존 MAP 기반 최적화가 "filtered" 객체 창이 groundtruth 객체와 잘 일치하지 못한다는 한계를 언급하지만, 본 논문에서 이 문제를 어떻게 완전히 극복했는지에 대한 심층적인 분석은 부족할 수 있다.
- **데이터셋 크기**: 새로운 데이터셋이 구축되었지만, 1000개의 이미지라는 크기는 딥러닝 모델 훈련에 있어 매우 큰 규모라고 보기는 어려울 수 있으며, 이는 모델의 일반화 성능에 영향을 미칠 수 있다.

## 6. 결론

이 논문은 현저한 영역 내에서 개별 객체 인스턴스를 식별하는 새로운 과제인 instance-level salient object segmentation을 제안하고 해결한다. 저자들은 하향식 및 상향식 정보를 통합하고 다중 스케일 특징 융합을 위한 어텐션 메커니즘을 사용하는 심층 다중 스케일 현저성 개선 네트워크(MSRNet)를 개발하여 견고한 현저한 영역 및 윤곽선 감지를 가능하게 한다. 추가적으로, 다중 스케일 조합 그룹화, MAP 기반 부분 집합 최적화 및 완전 연결 CRF 모델을 활용하여 인스턴스 제안을 정제한다. 이 연구는 현저한 인스턴스 분할을 위한 새로운 데이터셋을 구축하고, 공개 벤치마크 및 새 데이터셋에서 최첨단 결과를 달성함으로써 그 중요성과 효과를 입증한다. 이는 다양한 비전 애플리케이션에서 더 세부적인 이미지 이해를 위한 중요한 발판을 마련한다.
