# Recurrent Instance Segmentation

- **저자**: Bernardino Romera-Paredes, Philip Hilaire Sean Torr
- **발표연도**: 2015
- **arXiv**: https://arxiv.org/abs/1511.08250v3

## 1. 논문 개요

이 논문은 이미지 내에서 관심 있는 각 객체를 탐지하고 그 경계를 명확히 하는 문제인 인스턴스 분할(Instance Segmentation)에 대한 새로운 패러다임을 제안합니다. 기존 인스턴스 분할 접근 방식은 독립적으로 훈련된 여러 모듈의 앙상블로 구성되어 있어, 공동 학습의 기회를 놓쳤습니다. 본 연구는 인스턴스를 순차적으로 분할하는 방법을 학습하는 종단 간(end-to-end) 방법을 제안하여 이러한 문제를 해결합니다.

이 연구의 주요 목표는 인스턴스 분할 문제를 순차적으로 해결하는 새로운 접근 방식을 개발하는 것입니다. 이 문제의 중요성은 다양한 실제 응용 분야에 있습니다. 예를 들어, 로봇이 특정 객체를 잡기 위해 분할하거나, 부분적으로 시력이 약한 사람들을 위해 객체의 윤곽을 강조하고 향상시키는 데 유용합니다. 또한, 이미지 내 요소 개수 세기(예: 산업 공정에서 생산된 요소 수, 시위에 참석한 사람 수, 혈액 샘플에서 감염 및 건강한 혈액 세포 수 세기)와 같이 다양한 응용 분야에서 중요합니다. 인스턴스 분할은 픽셀별 분류를 다루는 의미론적 분할(Semantic Segmentation)보다 더 도전적인 문제로, 인스턴스의 개수가 사전에 알려지지 않기 때문에 더 복잡합니다.

## 2. 핵심 아이디어

이 논문의 핵심 아이디어는 순환 신경망(Recurrent Neural Network, RNN)을 기반으로 하여 이미지를 순차적으로 분석하고 한 번에 하나의 객체 인스턴스와 그 분할을 찾아내는 방식입니다. 이 모델은 공간 메모리를 사용하여 이미 설명된 픽셀을 추적하고 가려짐(occlusion)을 처리할 수 있도록 합니다. 이는 인간이 장면의 요소를 순차적으로 세고 정확한 공간 메모리를 사용하여 계산된 위치를 추적하는 방식에서 영감을 받았습니다.

기존 접근 방식은 종종 독립적으로 훈련된 여러 모듈로 구성되어 각 모듈이 다른 모듈의 출력에 적응하는 것을 배우지 못하고, 각 모듈에 대한 독립적인 손실 함수를 정의해야 하는 단점이 있었습니다. 본 연구는 이러한 단점을 피하기 위해 전체 인스턴스 분할 프로세스를 학습할 수 있는 종단 간 모델을 개발했습니다. 특히, 이 연구는 분류 단계를 제외하고 동일한 클래스에 속하는 인스턴스를 분할하는 '클래스별(class-specific)' 인스턴스 분할 문제에 중점을 둡니다.

## 3. 상세 방법 설명

![Figure 1: Diagram of Recurrent Instance Segmentation (RIS).](https://ar5iv.labs.arxiv.org/html/1511.08250/assets/x1.png)

제안된 접근 방식의 추론 과정은 그림 1에 설명되어 있으며 다음과 같습니다. 높이 $h$, 너비 $w$, 채널 $c$를 갖는 이미지 $\mathbf{I} \in \mathbb{R}^{h \times w \times c}$는 완전 컨볼루션 네트워크(Fully Convolutional Network)의 입력으로 사용됩니다. 이 네트워크는 이미지의 내부 표현에서 공간 정보를 보존하는 일련의 컨볼루션 및 최대 풀링 레이어로 구성됩니다. 이 네트워크의 출력인 $\mathbf{B} \in \mathbb{R}^{h' \times w' \times d}$는 각 픽셀에 대해 추출된 $d$차원 특징을 나타냅니다. 이 특징 맵의 크기는 입력 이미지보다 작을 수 있습니다($h' \leq h$, $w' \leq w$).

이 출력 $\mathbf{B}$는 시퀀스의 모든 반복에서 RNN의 입력이 됩니다. 시퀀스 시작 시, RNN의 초기 내부 상태 $\mathbf{h_0}$는 0으로 초기화됩니다. 첫 번째 반복 후, RNN은 이미지 내 인스턴스 중 하나의 분할과 예측의 신뢰도를 알려주는 지표(중지 조건으로 사용)를 생성합니다. 동시에 RNN은 최근 분할된 인스턴스를 설명하기 위해 내부 상태 $\mathbf{h_1}$을 업데이트합니다. 그런 다음 $\mathbf{B}$를 다시 입력으로, $\mathbf{h_1}$을 내부 상태로 사용하여 모델은 다른 분할된 인스턴스와 해당 신뢰도 점수를 출력합니다. 이 과정은 신뢰도 점수가 특정 수준 이하로 떨어질 때까지 반복되며, 이상적으로는 이미지의 모든 인스턴스를 분할하게 됩니다.

모델의 순차적 특성은 일반적인 인스턴스 분할 문제를 다룰 수 있게 합니다. 특히, 먼저 가려지지 않은 인스턴스를 분할하고, 이미 분할된 이미지 영역을 상태에 유지하여 가려진 객체를 탐지함으로써 가려짐을 암묵적으로 모델링할 수 있습니다. 상태의 또 다른 목적은 모델이 이미지 내의 다른 인스턴스 간의 잠재적 관계를 고려할 수 있도록 하는 것입니다.

이 접근 방식의 구조는 완전 컨볼루션 네트워크, RNN, RNN의 상태를 인스턴스 분할 및 신뢰도 점수로 변환하는 함수, 그리고 예측의 품질을 평가하고 최적화하려는 손실 함수로 구성됩니다.

### 3.1 컨볼루션 LSTM

Long Short-Term Memory (LSTM) 네트워크는 기울기 소실(vanishing gradient) 문제를 방지할 수 있는 능력 때문에 다른 순환 구조보다 뛰어나며, 많은 선행 연구에서 뛰어난 결과를 달성했습니다. 본 논문에서는 LSTM 유닛을 기반으로 하지만, 문제의 특성에 맞게 구조를 일부 변경했습니다.

모델의 입력과 출력이 모두 격자(특히 이미지)로부터의 맵이라는 점에서, 의미론적 분할이나 광학 흐름(optical flow)과 같은 문제는 종종 컨볼루션을 기반으로 하는 구조를 사용하여 해결됩니다. 이러한 구조에서 이미지의 중간 표현은 공간 정보를 보존합니다. 본 연구에서는 순환 유닛의 내부 상태 또한 공간 정보를 보존하는 맵으로 볼 수 있다는 점에 착안하여, RNN의 컨볼루션 버전, 특히 컨볼루션 장단기 기억(ConvLSTM) 유닛을 사용합니다.

ConvLSTM 유닛은 LSTM 유닛과 유사하지만, 각 게이트의 완전 연결 레이어가 컨볼루션으로 대체된다는 점이 유일한 차이점입니다. 업데이트 방정식은 다음과 같습니다:

$$
\begin{aligned}
\mathbf{i_{t}} &= {\rm Sigmoid}\left({\rm Conv}\left(\mathbf{x_{t}};\mathbf{w_{xi}}\right)+{\rm Conv}\left(\mathbf{h_{t-1}};\mathbf{w_{hi}}\right)+\mathbf{b_{i}}\right) \\
\mathbf{f_{t}} &= {\rm Sigmoid}\left({\rm Conv}\left(\mathbf{x_{t}};\mathbf{w_{xf}}\right)+{\rm Conv}\left(\mathbf{h_{t-1}};\mathbf{w_{hf}}\right)+\mathbf{b_{f}}\right) \\
\mathbf{o_{t}} &= {\rm Sigmoid}\left({\rm Conv}\left(\mathbf{x_{t}};\mathbf{w_{xo}}\right)+{\rm Conv}\left(\mathbf{h_{t-1}};\mathbf{w_{ho}}\right)+\mathbf{b_{o}}\right) \\
\mathbf{g_{t}} &= {\rm Tanh}\left({\rm Conv}\left(\mathbf{x_{t}};\mathbf{w_{xg}}\right)+{\rm Conv}\left(\mathbf{h_{t-1}};\mathbf{w_{hg}}\right)+\mathbf{b_{g}}\right) \\
\mathbf{c_{t}} &= \mathbf{f_{t}}\odot\mathbf{c_{t-1}}+\mathbf{i_{t}}\odot\mathbf{g_{t}} \\
\mathbf{h_{t}} &= \mathbf{o_{t}}\odot{\rm Tanh}(\mathbf{c_{t}})
\end{aligned}
$$

여기서 $\mathbf{x_t}$는 현재 시점의 입력, $\mathbf{h_{t-1}}$는 이전 시점의 숨겨진 상태, $\mathbf{c_{t-1}}$는 이전 시점의 셀 상태를 나타냅니다. $\mathbf{i_t}, \mathbf{f_t}, \mathbf{o_t}, \mathbf{g_t}$는 각각 입력, 망각, 출력, 게이트 게이트를 의미하며, Sigmoid와 Tanh 함수가 활성화 함수로 사용됩니다. $\odot$는 요소별 곱셈을 나타냅니다.

## 4. 실험 및 결과

이 논문에서는 제안된 모델의 성능을 평가하기 위해 두 가지 주요 실험을 수행했습니다.

1. **다중 인물 분할(Multiple Person Segmentation)**: 이 실험에서는 이미지 내의 여러 인물을 개별적으로 분할하는 작업을 수행했습니다.
2. **식물 잎 분할 및 개수 세기(Plant-leaves segmentation and counting)**: 이 실험에서는 식물 표현형 데이터셋(Plant Phenotyping dataset)을 사용하여 식물 잎을 분할하고 그 수를 세는 작업을 수행했습니다.

실험 결과, 제안된 방법은 다중 인물 분할에서 최근 접근 방식을 능가하는 성능을 보였습니다. 또한 식물 잎 개수 세기 작업에서는 모든 최신 기술(state-of-the-art) 접근 방식보다 뛰어난 성능을 달성했습니다. 이는 모델이 복잡한 인스턴스 분할 문제에 효과적이며, 특히 순차적 처리와 공간 메모리 활용이 중요한 이점을 제공함을 시사합니다.

## 5. 강점, 한계

**강점**:

- **종단 간 학습(End-to-End Learning)**: 기존의 독립적으로 훈련된 모듈 앙상블 방식과 달리, 전체 인스턴스 분할 프로세스를 종단 간으로 학습하여 공동 학습 기회를 활용하고 각 모듈 간의 불일치 문제를 해결합니다.
- **순차적 인스턴스 분할**: 인간의 인지 방식과 유사하게 인스턴스를 순차적으로 찾아 분할하며, 이를 통해 이미지 내 인스턴스 간의 순서 문제를 모델 스스로 결정할 수 있게 합니다.
- **공간 메모리 및 가려짐 처리**: RNN 기반의 공간 메모리를 통해 이미 설명된 픽셀을 추적하고, 이를 활용하여 가려진(occluded) 객체를 효과적으로 처리할 수 있습니다. 비가려진 인스턴스를 먼저 분할하고 그 정보를 상태에 저장함으로써 가려진 객체를 탐지합니다.
- **원칙적인 손실 함수**: 인스턴스 분할 문제의 속성을 정확하게 나타내는 원칙적인 손실 함수를 설계하여 모델 훈련의 안정성과 효율성을 높였습니다.
- **클래스별 인스턴스 분할에 효과적**: 특정 클래스 내의 인스턴스 분할 문제(예: 다중 인물 분할, 식물 잎 개수 세기)에서 뛰어난 성능을 보였습니다.

**한계**:

- **클래스별 인스턴스 분할에 초점**: 본 연구는 '클래스별(class-specific)' 인스턴스 분할에 중점을 두어 분류 단계를 제외하고 있습니다. 이는 다양한 종류의 인스턴스를 동시에 분할하고 분류하는 일반적인 의미론적 인스턴스 분할 문제에 직접적으로 적용하기 위해서는 추가적인 확장이 필요함을 의미합니다.
- **깊이 정보 활용의 한계**: 기존의 깊이 정보를 활용하여 인스턴스의 순서를 추론하는 접근 방식들이 인스턴스가 비슷한 깊이에 나타나는 경우(예: 스포츠 팀의 많은 선수들) 이점이 적다는 언급이 있습니다. 본 모델은 순차적 분할을 통해 이 문제를 간접적으로 해결하려 하지만, 깊이 정보를 명시적으로 활용하는 방식에 비해 특정 시나리오에서의 성능 비교는 추가 연구가 필요할 수 있습니다.

## 6. 결론

이 논문은 인스턴스 분할 문제에 대한 새로운 종단 간 순환 접근 방식인 Recurrent Instance Segmentation (RIS)을 제안합니다. RIS는 순환 신경망과 공간 메모리를 활용하여 이미지에서 인스턴스를 순차적으로 탐지하고 분할합니다. 이러한 접근 방식은 기존의 독립적인 모듈 기반 방법론의 한계를 극복하고, 공동 학습의 이점을 활용합니다.

주요 기여 사항은 다음과 같습니다:

1. 컨볼루션 레이어를 포함하는 RNN 기반의 종단 간 클래스별 인스턴스 분할 접근 방식 개발.
2. 인스턴스 분할 문제의 속성을 정확하게 나타내는 원칙적인 손실 함수 도출.

실험 결과, 제안된 방법은 다중 인물 분할에서 최신 접근 방식을 능가하고, 식물 표현형 데이터셋의 잎 개수 세기에서 모든 최신 기술 접근 방식을 뛰어넘는 성능을 입증했습니다. 이는 RIS 모델이 실제 적용(예: 로봇 공학, 의료 진단, 농업) 및 향후 연구(예: 더 일반적인 의미론적 학습 시스템으로 확장)에 중요한 역할을 할 잠재력을 가지고 있음을 시사합니다.
