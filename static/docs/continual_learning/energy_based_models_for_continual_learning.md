# Energy-Based Models for Continual Learning

**저자**: Shuang Li, Yilun Du, Gido M. van de Ven, Igor Mordatch

**연도**: 2020 (2022년 *Proceedings of The 1st Conference on Lifelong Learning Agents*, PMLR 199: 1‑22에 게재)

**arXiv ID**: <https://arxiv.org/abs/2011.12216v3>

---

## 1. 연구 배경 및 동기

연속 학습(Continual Learning, CL)은 신경망이 **재앙적 망각(catastrophic forgetting)** 없이 연속적인 작업들을 학습하도록 하는 것이 목표이다. 기존 방법들은 크게 **메모리 재생**, **정규화**, **아키텍처 확장**으로 나뉘지만 각각 한계가 있다:

* **메모리 재생**: 과거 데이터를 저장해야 하는데, 프라이버시나 저장 용량 문제로 실용적이지 않을 수 있다.
* **정규화**(예: EWC): 작업 간 차이가 클 경우 성능이 급격히 떨어진다.
* **아키텍처 확장**: 모델 크기와 연산 비용이 증가한다.

**에너지 기반 모델(Energy‑Based Models, EBMs)** 은 에너지 함수를 통해 데이터에 대한 정규화되지 않은 확률 분포를 정의한다. 에너지 지형을 조절함으로써 작업별 영역을 분리시켜, 명시적인 재생 없이도 작업 간 간섭을 감소시킬 수 있다.

본 논문은 **EBM을 연속 학습의 핵심 학습 패러다임**으로 활용한다. 이전에 학습된 데이터를 잘 보존하는 에너지 함수를 학습하면, 새로운 작업에 효율적으로 적응하면서도 과거 지식을 자연스럽게 유지할 수 있다고 가정한다.

---

## 2. 주요 기여

1. **EBM 기반 연속 학습 프레임워크** – 연속 학습을 에너지 최소화 문제로 정의하고, 현재 작업에 대해 에너지 함수를 학습하면서 이전 데이터의 에너지는 낮게 유지하도록 정규화한다.
2. **대조 발산(Contrastive Divergence) 기반 학습 목표** – 현재 작업의 샘플은 서로 가깝게, 이전 작업의 샘플은 멀리 떨어지게 하는 대조 손실을 도입해 재앙적 망각을 감소시킨다.
3. **작업 비의존 확장** – 작업 경계가 명시되지 않은 상황에서도 모델이 분포 변화를 감지하고 에너지를 조정하도록 확장한다.
4. **실험적 검증** – Split‑MNIST, Split‑CIFAR‑100 등 벤치마크에서 기존 강력한 베이스라인을 능가함을 보여주며, 감독·비감독 설정 모두에서 좋은 성능을 기록한다.

## 3. 방법론 요약

### 3.1 에너지 기반 모델 정의

EBM은 입력 $x$에 대해 에너지 함수 $E_{\theta}(x)$를 정의한다. 모델은 다음과 같은 확률을 할당한다:

$$
p_{\theta}(x) = \frac{e^{-E_{\theta}(x)}}{Z_{\theta}}
$$

여기서 $Z_{\theta}$는 정규화 상수(분할 함수)이며, 직접 계산하기 어려워 샘플링으로 근사한다.

### 3.2 연속 학습 목표

새로운 작업 $\mathcal{T}_t$와 데이터 $\mathcal{D}_t$에 대해 손실은 두 부분으로 구성된다:

* **작업 손실**: $\mathcal{D}_t$에 대한 최대우도(또는 대조) 손실.
* **정규화 손실**: 이전 작업에서 얻은 **버퍼** 샘플(또는 생성 재생 분포)에 대한 에너지 증가를 벌점으로 부과.

수식으로는 다음과 같다:

$$
\mathcal{L}\_t = \underbrace{\mathbb{E}\_{x\sim\mathcal{D}\_t}[E_{\theta}(x)]}\_{\text{작업 손실}} + \lambda \underbrace{\mathbb{E}\_{x\sim\mathcal{B}\_{<t}}[\max(0, E_{\theta}(x) - m)]}\_{\text{망각 페널티}}
$$

여기서 $\mathcal{B}_{<t}$는 이전 작업의 저장 혹은 생성 샘플, $\lambda$는 두 항의 가중치, $m$는 마진이다.

### 3.3 대조 발산(Contrastive Divergence) 학습

저자들은 **CD‑k** 를 이용해 로그우도 그래디언트를 근사한다. 짧은 마코프 체인을 데이터 포인트에서 시작해 샘플링한다. CD 손실은 자연스럽게 **음성 샘플의 에너지를 높이고, 양성 샘플의 에너지를 낮추는** 대조 항을 제공한다.

### 3.4 작업 비의존 탐지

작업 경계가 주어지지 않을 경우, 모델은 **입력 배치의 평균 에너지**를 모니터링한다. 평균 에너지가 급격히 상승하면 분포 변화로 판단하고, 새로운 작업 전용 에너지 항을 추가하면서 기존 항은 고정한다.

## 4. 실험 및 결과

| 데이터셋 | 평가 지표 | Baseline (EWC) | Baseline (Replay) | **EBM (제안 방법)** |
|----------|----------|----------------|-------------------|---------------------|
| Split‑MNIST | 평균 정확도 | 78.3% | 92.1% | **94.5%** |
| Split‑CIFAR‑100 | 평균 정확도 | 45.2% | 61.8% | **66.3%** |
| Permuted‑MNIST | 평균 정확도 | 70.1% | 88.4% | **90.2%** |

EBM 방식은 일관되게 **높은 정확도**와 **낮은 망각률**을 보였다. Ablation 실험에서는 대조 항이 성능 향상에 가장 크게 기여했으며, 작업 비의존 버전은 오라클 버전 대비 약 2% 정도만 성능이 떨어졌다.

## 5. 한계 및 향후 연구

* **샘플링 비용**: CD‑k 는 고차원 데이터에서 MCMC 단계를 필요로 하여 계산 비용이 크다.
* **확장성**: 현재 실험은 이미지 분류에 국한돼 있어, 대규모 비전·언어 모델에 적용하려면 보다 효율적인 샘플링(예: score‑based 모델)이 필요하다.
* **작업 탐지 민감도**: 단순 에너지 임계값 방식은 점진적인 도메인 이동을 오탐지할 수 있다. 더 정교한 드리프트 탐지 기법이 요구된다.
* **미래 연구 방향**
  1. **고급 샘플러**(예: 학습된 프리컨디셔너를 갖는 Langevin dynamics) 도입으로 오버헤드 감소.
  2. **연속 생성 모델링**으로 EBM 자체가 재생 데이터를 생성하도록 확장.
  3. **아키텍처 성장**과 결합해 매우 많은 작업을 처리할 수 있는 프레임워크 구축.

---

## 6. 실무자를 위한 핵심 인사이트

* **에너지 형태 조정**은 원본 데이터를 저장하지 않고도 과거 지식을 보존할 수 있어 프라이버시‑민감한 환경에 유리하다.
* 대조 발산 목표는 기존 학습 파이프라인에 **소폭의 수정만으로** 적용 가능하다.
* 입력 배치의 평균 에너지를 모니터링하는 것은 **작업 변화 감지**를 위한 가벼운 모델 내재 신호이다.
