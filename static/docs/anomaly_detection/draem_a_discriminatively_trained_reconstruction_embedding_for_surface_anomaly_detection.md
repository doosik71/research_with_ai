# DRAEM - A discriminatively trained reconstruction embedding for surface anomaly detection

- **저자**: Vitjan Zavrtanik, Matej Kristan, Danijel Skočaj
- **발표연도**: 2021
- **arXiv**: https://arxiv.org/abs/2108.07610

## 1. 논문 개요

### 논문의 목표

본 논문의 목표는 표면 이상치 탐지(surface anomaly detection)를 위한 새로운 딥러닝 방법론인 DRÆM(Discriminatively trained Reconstruction Anomaly Embedding Model)을 제안하는 것이다. 이 방법은 비정상적인 외관을 가진 이미지 영역의 위치를 정확하게Localisation(지역화)하는 것을 목적으로 한다.

### 연구 문제

표면 이상치 탐지는 정상적인 외관에서 벗어나는 이미지 영역의Localisation을 다루는 문제이다. 품질 관리 및 표면 결함Localisation 응용 분야에서 흔히 발생하는 이 작업은 특히 도전적인데, 그 이유는 이미지 내 이상치가 아주 작은 부분만을 차지하고 있으며, 종종 훈련 데이터 분포에 가깝기 때문이다. 실제 응용에서는 이상치의 외관이 크게 달라질 수 있고, 이상치가 있는 이미지는 드물며, 수동 주석은 너무 많은 시간이 소요될 수 있다. 이로 인해 많은 경우 이상치가 없는 이미지만을 포함하는 극도로 불균형한 훈련 데이터셋이 형성된다.

### 문제의 중요성

산업 제조업에서의 품질 관리는 제품의 결함을 실시간으로 탐지하는 것이 필수적이다. 수작업으로 결함을 검사하는 것은 시간과 비용이 많이 들며, 일관된 품질 유지를 어렵게 한다. 자동화된 표면 이상치 탐지 방법은 제조 품질을 향상시키고 생산 비용을 절감하는 데 중요한 역할을 한다. 그러나 실제 환경에서는 다양한 유형의 결함이 발생할 수 있고, 이러한 결함에 대한 레이블이 충분히 확보되기 어려워 비지도 학습 기반 방법이 필요하다.

## 2. 핵심 아이디어

### 중심 직관

DRÆM의 핵심 아이디어는 재구성 접근 방식과 판별 접근 방식을 결합하는 것이다. 기존 재구성 기반 방법들은 정상 이미지로만 훈련되어 비정상 영역에서 Poor reconstruction을 하지만, 판별 기반 방법들은 합성 이상치에 과적합되어 실제 이상치에 일반화되지 못한다. 저자들은 원본 이미지와 재구성 이미지의 결합 공간에서 판별 모델을 학습함으로써, 합성 이상치의 외관에 과적합하지 않으면서도 실제 이상치에 잘 일반화되는 방법을 제안한다.

### 기존 접근 방식과의 차별점

기존 표면 이상치 탐지 방법들은 크게 두 가지 범주로 나뉜다. 첫째, Autoencoder와 GAN을 사용한 재구성 기반 방법들은 정상 데이터만으로 강력한 재구성 부분 공간을 학습하지만, 정상 외관과 유사한 이상치를 잘 재구성하여 탐지하지 못한다. 둘째, 합성 이상치로 훈련된 판별 방법은 합성 외관에 과적합되어 실제 이상치에 일반화되지 못한다. DRÆM은 이 두 접근 방식의 장점을 결합하여, 재구성 하위 네트워크와 판별 하위 네트워크를 End-to-End로 공동 훈련시킨다.

## 3. 상세 방법 설명

### 전체 파이프라인

DRÆM은 재구성 하위 네트워크와 판별 하위 네트워크의 두 부분으로 구성된다. 재구성 하위 네트워크는 입력 이미지에서 이상치를 암묵적으로 탐지하고 의미적으로 그럴듯한 정상 콘텐츠로 재구성한다. 판별 하위 네트워크는 재구성된 이미지와 원본 이미지를 채널 방향으로 연결한 결합 입력을 사용하여 정확한 픽셀 단위 이상치 탐지 맵을 생성한다.

### 재구성 하위 네트워크

재구성 하위 네트워크는 인코더-디코더 아키텍처로 формуле된다. 이 네트워크는 입력 이미지의 지역적 패턴을 정상 샘플의 분포에 더 가까운 패턴으로 변환한다. 네트워크는 인공적으로 손상된 버전의 이미지에서 원본 이미지를 재구성하도록 훈련된다.

재구성 손실 함수는 $l_2$ 손실과 SSIM(Structural Similarity) 손실의 조합으로 정의된다:

$$L_{SSIM}(I, I_r) = \frac{1}{N_p} \sum_{i=1}^{H} \sum_{j=1}^{W} 1 - SSIM(I, I_r)_{(i,j)}$$

$$L_{rec}(I, I_r) = \lambda L_{SSIM}(I, I_r) + l_2(I, I_r)$$

여기서 $I$는 입력 이미지, $I_r$는 재구성된 이미지, $\lambda$는 손실 균형 하이퍼파라미터이다.

### 판별 하위 네트워크

판별 하위 네트워크는 U-Net 유한 아키텍처를 사용한다. 입력 $I_c$는 재구성 하위 네트워크 출력 $I_r$와 입력 이미지 $I$를 채널 방향으로 연결한 것이다. 판별 하위 네트워크의 출력에 Focal Loss를 적용하여 어려운 예제의 정확한 분할에 대한 견고성을 높인다.

전체 손실 함수는 다음과 같이 정의된다:

$$L(I, I_r, M_a, M) = L_{rec}(I, I_r) + L_{seg}(M_a, M)$$

여기서 $M_a$는 정답 이상치 분할 마스크, $M$은 네트워크 출력 분할 마스크이다.

### 합성 이상치 생성

DRÆM은 실제 이상치의 외관을 현실적으로 반영할 필요 없이, 단순히 분포 외(out-of-distribution) 외관을 생성하여 적절한 거리 함수를 학습하는 것이 핵심이다. 이상치 시뮬레이터는 Perlin 노이즈 생성기로 다양한 이상치 형태를 생성하고, 이진화된 이상치 마스크를 만든다. 이상치 텍스처 소스 이미지는 입력 이미지 분포와 무관한 이상치 소스 이미지 데이터셋에서 샘플링되며, RandAugment에서 영감을 받은 무작위 증강이 적용된다.

증강된 훈련 이미지 $I_a$는 다음과 같이 정의된다:

$$I_a = \overline{M}\_a \odot I + (1 - \beta)(M_a \odot I) + \beta(M_a \odot A)$$

여기서 $\overline{M}\_a$는 이상치 마스크의 역이고, $\odot$는 요소별 곱셈 연산이며, $\beta$는 블렌딩의 불투명도 매개변수이다.

### 이미지 수준 이상치 점수 추정

판별 하위 네트워크의 출력은 픽셀 수준 이상치 탐지 마스크 $M_o$이다. 이미지에 이상치가 있는지 여부를 판단하기 위해, $M_o$를 평균 필터 컨볼루션 레이어로 스무딩한 후, 스무딩된 이상치 점수 맵의 최대값을 취하여 최종 이미지 수준 이상치 점수 $\eta$를 계산한다:

$$\eta = max(M_o * f_{s_f \times s_f})$$

## 4. 실험 및 결과

### 데이터셋

저자들은 MVTec anomaly detection 데이터셋에서 광범위한 평가를 수행하였다. 이 데이터셋은 15개의 객체 클래스와 다양한 이상치를 포함하여 표면 이상치 탐지 방법의 일반적인 평가를 가능하게 한다. 또한 DAGM 데이터셋에서도 평가하였다.

### 기준선 방법

비교를 위해 최근의 최첨단 방법들인 Uninformed Students, RIAD, PaDim, 그리고 다양한 Autoencoder 기반 방법들과 GAN 기반 방법들을 포함하였다.

### 표면 이상치 탐지 결과

MVTec 데이터셋에서 이미지 수준 표면 이상치 탐지 작업에서 DRÆM은 15개 클래스 중 9개에서 최고 AUROC를 달성하고, 나머지 클래스에서도 유사한 결과를 얻었다. 이전 최고 성능 방법보다 2.5 퍼센트 포인트 향상되었다. 평균 AUROC는 98.0%를 달성하여 이전 최고 성능인 95.5%를 능가하였다.

### 이상치Localisation 결과

픽셀 수준 이상치Localisation 작업에서 DRÆM은 AUROC 점수에서 이전 최고 성능 방법과 유사한 결과를 달성하면서, AP(average precision) 점수에서 13.4 퍼센트 포인트 향상되었다. 15개 클래스 중 11개에서 더 나은 AP 점수를 달성하였다.

### 완전 지도 학습 방법과의 비교

완전 지도 학습 방법들은 MVTec에서 평가할 수 없으므로, DAGM 데이터셋에서 완전 지도 학습 방법들과 비교하였다. DRÆM은 지도 학습 방법들과 유사한 분류 성능을 달성하면서,Localisation 정확도에서 이들을 능가하였다. 이는 DRÆM이 실제 이상치 없이 훈련되었음에도 불구하고 주목할 만한 결과이다.

### 소거 연구 결과

재구성 하위 네트워크의 기여도를 분석하기 위해 이를 제거하고 판별 하위 네트워크만 훈련한 결과, 합성 이상치에 과적합되어 성능이 저하되었다. 이는 재구성 하위 네트워크가 판별 하위 네트워크가 합성 외관에 과적합하는 것을 방지하는 데 중요함을 보여준다. 이상치 소스 데이터셋(DTD vs ImageNet)의 영향 분석 결과, 증강과 불투명도 무작위화가 상당한 기여를 하여 매우 적은 수의 텍스처 이미지(10개 미만)만으로도 최첨단 결과를 달성할 수 있었다.

## 5. 강점, 한계

### 강점

본 논문의 주요 강점은 다음과 같다. 첫째, 재구성과 판별 접근 방식을 결합하여 두 방법의 장점을 모두 활용한다. 재구성 하위 네트워크가 비정상 영역을 정상 콘텐츠로 채우고, 판별 하위 네트워크가 원본과 재구성 이미지의 결합 공간에서 이상치를 정확히 분할한다. 둘째, 합성 이상치만으로 훈련되어 실제 이상치 레이블이 필요 없다. 합성 외관이 실제 이상치를 현실적으로 반영할 필요 없이, 단순히 분포 외 패턴을 생성하면 된다. 셋째, 복잡한 후처리 없이 직접 이상치Localisation이 가능하다. 넷째, 다양한 텍스처 소스 데이터셋에서 효과적으로 작동하며, 극단적인 경우 단색 영역만으로도 최첨단 결과를 달성한다.

### 한계 및 미해결 질문

본 논문에는 몇 가지 한계가 존재한다. 첫째, 객체의 일부가 없는情况和 같이 정상 분포에 매우 가까운 이상치를 탐지하기 어렵다. 이러한 경우 결손된 영역에 일반적으로 나타나는 다른 피처가 포함되어 있어 정상 영역과 구별하기 어렵다. 둘째, 모호한 이상치에 대한 정답 레이블이 부정확할 수 있다. 전체 표면을 이상지로 표시하지만 실제로는 일부만 이상인 경우가 있다. 셋째, 훈련을 위해 이상치가 없는 이미지가 필요하며, 이는 일부 실제 시나리오에서 제한이 될 수 있다.

## 6. 결론

본 논문은 표면 이상치 탐지 및Localisation을 위한 판별적 End-to-End 훈련 가능한 방법 DRÆM을 제안하였다. MVTec 데이터셋에서 표면 이상치 탐지 작업에서 기존 최첨단 대비 2.5 AUROC 포인트,Localisation 작업에서 13.5 AP 포인트를 향상시켰다. DAGM 데이터셋에서는 완전 지도 학습 방법에 가까운 분류 성능을 달성하면서Localisation 정확도에서 이를 능가하였다.

재구성 하위 네트워크를 통한 결합 재구성-이상치 임베딩 학습 패러다임이 표준 방법보다 결과를 크게 향상시키며, 합성 이상치에 대한 과적합 없이 실제 이상치에 잘 일반화됨을 입증하였다.
