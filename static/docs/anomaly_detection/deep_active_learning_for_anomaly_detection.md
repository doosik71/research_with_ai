# Deep Active Learning for Anomaly Detection

- **저자**: Tiago Pimentel, Marianne Monteiro, Adriano Veloso, Nivio Ziviani
- **발표연도**: 2018
- **arXiv**: https://arxiv.org/abs/1805.09411

## 1. 논문 개요

본 논문의 목표는 비지도 이상 감지(unsupervised anomaly detection)에 능동 학습(active learning)을 결합하여 이상 데이터와 정상 데이터를 효과적으로 분리하는 방법을 제안하는 것이다. 연구의 핵심 동기는 이상 감지 응용에서 레이블 취득이 비용이 많이 들고 시간이 소요되지만, 인간 전문가의 피드백을 통해 필요한 사전 정보를 제공할 수 있다는 점이다.

비지도 이상 감지는 레이블이 없는 데이터에 적용되므로 암시적 사전 분포(implicit priors)나 휴리스틱을 사용하여 이상을 발견하고 각 인스턴스에 대한 이상 점수(anomaly score)를 제공한다. 그러나 이러한 접근법은 이상 분포에 대한 사전 정보를 필요로 하며, 이 없이는 완벽한 성능 보장을 제공하기 어렵다. 본 논문에서는 기존 비지도 이상 감지 딥러닝 모델에 UAI(Unsupervised to Active Inference) 레이어를 부착하여 능동 학습 방법론으로 변환하는 새로운 레이어를 도입한다.

## 2. 핵심 아이디어

논문의 중심 아이디어는 기존 비지도 이상 감지 딥러닝 모델에 간단하게 부착할 수 있는 새로운 UAI 레이어를 제안하는 것이다. 이 레이어는 학습된 잠재 표현(learned latent representations)과 이상 점수를 결합하여 분류기를 학습하며, 전문가의 피드백을 통해 이상 감지 성능을 점진적으로 개선한다.

기존 접근 방식과의 주요 차별점은 다음과 같다. 첫째, 기존 비지도 방법들은 저차원 모델은 군집된 이상에 높은 점수를 부여하고 고차원 모델은 저밀도(low density) 이상에 높은 점수를 부여하는 등 모델 용량(model capacity)에 따라 탐지되는 이상 유형이 달라진다. UAI 레이어는 능동 학습을 통해 이 문제를 해결한다. 둘째, 기존 semi-supervised 방법들은 정상 데이터만으로 학습하지만, UAI는 소수의 능동적으로 레이블된 인스턴스를 활용하여 더 나은 결정 경계를 학습한다. 셋째, Gradient flow가 잠재 표현 $l$을 통해 흐를 수 있도록 허용하여 네트워크가 이상 공간에서 데이터를 더 잘 분리하도록 돕는다.

## 3. 상세 방법 설명

### 3.1 문제 정의

논문에서는 이상 감지 문제를 확률론적 관점에서 정의한다. 정상 데이터는 확률 밀도 함수 $p_{\text{normal}}(x) = p(x \mid y=0)$에서 샘플링되고, 이상 데이터는 $p_{\text{anom}}(x) = p(x \mid y=1)$에서 샘플링된다고 가정한다. 전체 데이터셋은 다음의 혼합 분포에서 샘플링된다:

$$p_{\text{full}}(x) = (1-\lambda)p_{\text{normal}}(x) + \lambda p_{\text{anom}}(x)$$

여기서 $\lambda$는 임의의 데이터 포인트가 이상일 확률을 나타내는 작은 상수이다. 일반적인 혼합 모델은 식별 불가능(unidentifiable)하다는 잘 알려진 결과가 있으므로, $\lambda$가 작은 경우 $p_{\text{full}}$만으로는 $p_{\text{anom}}$에 대한 정보를 얻을 수 없으며, 이는 비지도 이상 감지가 $p_{\text{anom}}$에 대한 사전 정보를 필요로 함을 의미한다.

### 3.2 UAI 레이어 아키텍처

UAI 레이어는 두 가지 입력을 받는다: 모델이 생성한 잠재 표현 $l(x)$와 출력 이상 점수 $s(x)$이다. 이를 분류기에 입력하여 각 인스턴스의 이상 확률을 다음과 같이 정의한다:

$$\hat{p}(y \mid x) \propto s_{\text{uai}}(x) = \text{classifier}([l(x); s(x)])$$

본 논문에서는 로지스틱 회귀를 분류기로 사용하며, 구체적인 수식은 다음과 같다:

$$\hat{p}(y \mid x) \propto s_{\text{uai}}(x) = \sigma(W_{\text{act}}[l(x); s(x)] + b_{\text{act}})$$

여기서 $W_{\text{act}} \in \mathbb{R}^{1,d+1}$는 선형 변환, $b_{\text{act}} \in \mathbb{R}$는 편향항, $\sigma(\cdot)$는 시그모이드 함수이다. 분류기는 크로스 엔트로피 손실 함수로 역전파를 통해 학습되며, 그래디언트가 $l$을 통해 흐르지만 $s$를 통해 흐르지 않도록 허용한다.

### 3.3 기본 모델: Denoising Autoencoder

첫 번째 기본 모델은 Denoising Auto-Encoder(DAE)이다. 인코더가 입력을 잠재 공간으로 변환하고, 디코더가 이 잠재 표현을 사용하여 입력을 재구성한다. 손실 함수는 재구성 오류의 $L_2$ 노름을 최소화한다:

$$l = f_{\text{enc}}(x + \epsilon), \quad \epsilon \sim \mathcal{N}(0, \varphi)$$

$$\hat{x} = f_{\text{dec}}(l)$$

$$\mathcal{L}\_{\text{DAE}} = \|x - \hat{x}\|\_2^2$$

이상 점수는 재구성 오류로 정의된다:

$$s_{\text{DAE}}(x) = \|x - \hat{x}\|\_2^2$$

DAE에 UAI 레이어를 부착한 $DAE_{\text{uai}}$는 다음과 같이 정의된다:

$$l_{\text{DAE}} = f_{\text{enc}}(x + \epsilon)$$

$$s_{\text{DAE-uai}}(x) = \text{uai}([l_{\text{DAE}}; s_{\text{DAE}}])$$

### 3.4 기본 모델: Multi-Layer Perceptron

두 번째 기본 모델은 분류기를 학습시킨 MLP이다. 인스턴스의 클래스 레이블을 다른 특성에서 예측하도록 학습하며, 데이터 포인트의 크로스 엔트로피가 이상 점수의 추정치로 사용된다:

$$\widehat{x}\_y = f_{\text{Class}}(x_x)$$

$$\mathcal{L}\_{\text{Class}} = H(x_y, \widehat{x}\_y) = \text{cross\_entropy}(x_y, \widehat{x}\_y)$$

$$s_{\text{Class}}(x) = H(x_y, \widehat{x}\_y)$$

이 모델에서 마지막 숨겨진 레이어($h_{p-1}$)가 잠재 표현으로 사용된다.

### 3.5 전체 손실 함수

UAInets를 학습하는 전체 손실 함수는 다음과 같다:

$$\mathcal{L}\_{\text{uai}} = \mathbf{1} \cdot H(y, s_{\text{uai}}(x))$$

$$\mathcal{L}\_{\text{full}} = \mathcal{L}\_{\text{uai}} + \mathcal{L}\_{\text{base}}$$

여기서 $\mathbf{1}$은 이미 능동적으로 레이블된 인스턴스에 대해 값이 1이고 레이블이 없는 인스턴스에 대해 0인 지시 함수이다. $\mathcal{L}\_{\text{base}}$는 기본 네트워크의 손실 함수로, $\mathcal{L}\_{\text{DAE}}$ 또는 $\mathcal{L}\_{\text{Class}}$이다.

### 3.6 능동 이상 감지 알고리즘

알고리즘의 전체 흐름은 다음과 같다. 먼저, 전체 데이터셋 $\mathcal{D}$로 기본 모델(DAE 또는 Class)을 5,000 스텝 동안 사전 학습한다. 그런 다음 각 반복에서 상위 $k=10$개의 데이터 포인트(가장 의심스러운 것)를 전문가에게 레이블링하도록 보내고, 레이블이 반환되면 전체 모델을 100 스텝 동안 추가 학습시킨다. 이 과정을 예산 $b$가 소진될 때까지 반복한다.

## 4. 실험 및 결과

### 4.1 합성 데이터 실험

MNIST 데이터셋을 기반으로 세 가지 유형의 합성 이상 감지 데이터셋을 구성하여 실험을 수행하였다.

MNIST0: 숫자 '0'의 빈도를 원래의 10%로 줄이고, 남은 '0'들의 레이블을 무작위로 [1;9] 중 하나로 변경하여 이상으로 정의하였다. 이상은 전체 데이터의 약 1.1%를 차지한다.

MNIST0-2: 숫자 '0', '1', '2'의 빈도를 줄이고 레이블을 [3;9] 중 하나로 변경하였다. 이상은 전체의 약 4.1%이다.

MNISThard: 약한 1계층 분류기로 MNIST를 학습시킨 후, 이 분류기에서 잘못 분류된 모든 데이터 포인트를 이상으로 정의하였다. 이상은 약 3.3%이다.

결과 분석에서 Class 모델은 MNIST0과 MNIST0-2(군집된 이상)에서 좋은 결과를 얻었지만 MNISThard(저밀도 이상)에서는 성능이 저하되었다. 반면 DAE 모델은 MNISThard에서는 좋은 결과를 얻었지만 MNIST0과 MNIST0-2에서는 성능이 저조하였다. 그러나 두 가지 UAI 네트워크(DAEuai와 Classuai)는 세 가지 모든 데이터셋에서 강건한(robust) 성능을 보였다.

### 4.2 Fashion-MNIST 실험

Fashion-MNIST 데이터셋에서도 동일한 절차로 실험을 수행하였다. 이 데이터셋에서는 이상치가 더 식별하기 어려운 것으로 나타났다. 특히 한 실험 실행에서 DAEuai는 능동 피드백을 시작하기 위해 여러 번의 반복이 필요하였으며, 처음 5,000번의 선택에서 정상 인스턴스만 발견하여 UAI 레이어가 개선할 신호가 없었다.

### 4.3 잠재 표현 시각화

학습 과정에서 잠재 표현과 이상 점수의 진화를 분석하였다. MNIST0-2와 MNISThard에서 1차원 잠재 표현($l(x) \in \mathbb{R}^1$)을 사용한 결과, 초기에는 이상과 정상 데이터가 분리되지 않았지만, 소수의 레이블된 인스턴스($b=250$)만으로 이상 공간이 훨씬 분리하기 쉽게 변화하였다. $b=2,000$에서는 이상 공간이 거의 완벽하게 선형적으로 분리 가능해졌다.

### 4.4 실제 데이터셋 실험

실제 이상 감지 벤치마크 데이터셋에서 성능을 평가하였다. KDDCUP, Arrhythmia, Thyroid, KDDCUP-Rev, Yeast, Abalone, CTG, Credit, Covtype, MNMG, Shuttle 등의 데이터셋을 사용하였다. 비교 대상으로 DAE, DAGMM, LODA-AAD, Tree-AAD를 포함하였다.

주요 결과는 DAE 자체는 모든 데이터셋에서 성능이 저조하였으나, DAEuai는 매우 간단한 아키텍처를 사용했음에도 불구하고 모든 데이터셋에서 최상의 성능(또는 최상에 가까운 성능)을 달성하였다. 특히 Covtype 데이터셋(286,048개 인스턴스 중 0.17% 이상)에서는 다른 경쟁 모델들과의 차이가 가장 크게 나타났다. DAEuai와 AAD는 모든 실험에서 동일한 하이퍼파라미터를 사용하였으며(예외적으로 이상 비율이 100개 미만인 데이터셋에서는 $k$를 10에서 3으로 줄임), 이는 하이퍼파라미터 튜닝 없이도 좋은 성능을 달성할 수 있음을 보여준다.

## 5. 강점, 한계

### 5.1 강점

본 논문의 강점은 다음과 같다. 다양한 유형의 이상에 대해 강건한 성능을 보인다. 저밀도 이상에는 DAE 기반 모델이, 군집된 이상에는 Class 기반 모델이 효과적이지만, UAI 레이어를 부착하면 두 유형 모두에서 좋은 성능을 달성한다. 하이퍼파라미터 튜닝이 필요 없다. 실제 응용에서는 (초기에) 튜닝할 레이블된 데이터가 없거나 너무 적어 신뢰할 수 없으므로, 이는 중요한 장점이다. 간단한 아키텍처에도 최상의 성능을 달성한다. 복잡한 구조 없이도 기존 최첨단 방법들과 동등하거나 더 나은 결과를 얻는다. End-to-end 학습이 가능하다. 전체 모델이 역전파를 통해 공동으로 학습되어 표현 학습과 이상 감지가 통합된다.

### 5.2 한계 및 미해결 질문

본 연구의 한계는 다음과 같다. 전문가 피드백의 품질에 의존한다. 잘못되거나 일관성 없는 피드백은 성능 저하로 이어질 수 있다. 일부 실험에서 UAI 네트워크가 초기 학습 단계에서 느리게 시작하는 현상이 관찰되었다. 이는 특히 학습된 표현이 초기에는 이상과 정상을 잘 분리하지 못하는 경우에 발생한다. 소량의 레이블된 데이터만으로도 좋은 성능을 보이지만, 극도로 드문 이상에 대한 효과성은 충분히 검증되지 않았다. 모델이 이상으로 분류될 확률을 직접 추정하지만, 이것이 실제로 의미하는 바에 대한 해석 가능성은 제한적이다.

## 6. 결론

본 논문의 주요 기여는 세 가지로 요약된다. 첫째, UAI(Unsupervised to Active Inference) 레이어를 제안하여, 어떤 비지도 이상 감지 딥러닝 모델에도 쉽게 부착할 수 있는 능동 학습 방법을 도입하였다. 둘째, 자동 인코더와 다층 퍼셉트론이라는 매우 간단한 아키텍처에도 최첨단 수준의 결과를 달성할 수 있음을 보여주었다. 셋째, 학습된 표현과 이상 점수의 강점을 결합하여 레이블된 인스턴스와 전체 데이터셋을 모두 활용하는 End-to-end 아키텍처를 제시하였다.

향후 연구 방향으로, GAN(Generative Adversarial Networks) 기반 방법과의 결합, 더 복잡한 시계열 이상 감지 문제로의 확장, 그리고 실제 산업 응용(예: 네트워크 침입 탐지, 사기 탐지 등)에서의 검증이 있다. 이 연구는 레이블 취득 비용이 높고 이상 데이터가 드문 실제 환경에서 이상 감지 시스템의 실용성을 크게 향상시킬 수 있는 잠재력을 보여준다.
