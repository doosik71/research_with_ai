# Robust Anomaly Detection in Images using Adversarial Autoencoders

- **저자**: Laura Beggel, Michael Pfeiffer, Bernd Bischl (Technical University of Munich)
- **발표연도**: 2019
- **arXiv**: https://arxiv.org/abs/1901.06355

## 1. 논문 개요

이 논문의 목표는 이미지에서 이상 탐지를 수행할 때, 학습 데이터에 소수의 이상 데이터가 포함되어 있더라도 효과적으로 동작하는 강건한 이상 탐지 방법을 제안하는 것이다. Autoencoder 기반 이상 탐지 방법은 학습 데이터를 정상 이미지로만 구성한다는 가정에 기반하지만, 실제 환경에서는 이러한 깨끗한 데이터셋을 보장하기 어렵다.

논문이 해결하려는 핵심 연구 문제는 기존 Autoencoder 기반 이상 탐지 방법이 학습 데이터에 포함된 소수의 이상에 매우 민감하다는 점이다. 학습 중 이상 데이터의 재구성 오류가 지속적으로 감소하여 최종적으로 정상 데이터와 이상 데이터의 재구성 오류 분포가Overlap되어 탐지 성능이 저하된다.

이 연구의 중요성은 시각적 품질 검사, 감시, 의료 영상 분석 등 실제 응용 분야에서 깨끗한 정상 데이터셋을 확보하기 어렵거나 비용이 많이 드는 상황에서, 오염된 데이터셋에서도 효과적으로 동작하는 방법이 필요하기 때문이다.

## 2. 핵심 아이디어

이 논문의 중심 아이디어는 **Adversarial Autoencoder (AAE)** 를 활용하여 잠재 공간에서의 우도(likelihood)를 추가로 고려하고, **Iterative Training Set Refinement (ITSR)** 를 통해 학습 데이터에서 이상 후보를 반복적으로 식별하고 제거하는 것이다.

기존 접근 방식과의 핵심 차별점은 다음과 같다. 기존 Autoencoder는 재구성 오류만 이상 점수로 사용하지만, 이 논문에서는 잠재 공간에서의 우도와 재구성 오류를 결합한 새로운 판정 기준을 제안한다. 또한 ITSR을 통해 학습 중 능동적으로 이상 데이터를 식별하고 제외함으로써, 학습 데이터 오염에 대한 강건성을 확보한다.

## 3. 상세 방법 설명

### 일반 Autoencoder의 한계

Autoencoder는 인코더 $f: \mathcal{X} \to \mathcal{Z}$와 디코더 $g: \mathcal{Z} \to \mathcal{X}$로 구성되며, 입력 이미지 $\mathbf{x}$를 저차원 잠재 표현 $\mathbf{z} = f(\mathbf{x})$로 변환한 후 복원한다. 재구성 오류 $L(\mathbf{x}, \mathbf{x}') = L(\mathbf{x}, g(f(\mathbf{x})))$를 이상 점수로 사용하며, 임계값 $T_{rec}$를 초과하면 이상으로 분류한다.

학습 데이터에 이상이 포함된 경우, SGD 기반 학습의 특성상 초기에는 정상 데이터의 재구성 오류가 빠르게 감소하지만, 이후 이상 데이터의 높은 오류가 더 큰 그래디언트를 생성하여 학습 방향을 지배하게 된다.이로 인해 학습 후반부에는 정상과 이상 데이터의 재구성 오류 차이가 사라진다.

### Adversarial Autoencoder (AAE)

AAE는 잠재 공간에 사전 분포 $p(\mathbf{z})$를 부과하여 일반 Autoencoder를 확장한다. 판별자(discriminator) 네트워크가 인코더의 출력과 사전 분포에서 샘플링된 값을 구분하도록 학습되며, 이를 통해 잠재 코드가 사전 분포를 따르도록 강제한다.

잠재 공간에서의 우도 $p(f(\mathbf{x}))$를 이상 점수로 사용할 수 있다. 정상 데이터는 높은 우도를, 이상 데이터는 낮은 우도를 가질 것으로 기대한다.

### 결합 판정 기준

재구성 오류와 잠재 우도를 결합한 판정 기준을 제안한다. 새로운 이미지 $\mathbf{y}$는 다음 조건 중 하나라도 만족하면 이상으로 분류된다.

$$L(\mathbf{y}, \mathbf{y}') > T_{rec} \quad \text{or} \quad p(f(\mathbf{y})) < T_{prior}$$

### Iterative Training Set Refinement (ITSR)

ITSR는 세 단계로 구성된다.

**1단계: Pre-training**
AAE를 전체 학습 데이터에서 일정 에폭만큼 학습하며, 모든 샘플 가중치를 $w_i = 1$로 설정한다.

**2단계: Detection and Refinement**
1-class SVM을 잠재 표현에 적용하여 이상 후보 $\hat{A}$를 식별한다. 1-class SVM의 파라미터 $\nu$는 예상 이상 비율의 상한선을 설정한다. 식별된 이상 후보의 가중치를 $w_i = 0$으로 설정하여 학습에서 제외한 후 모델을 재학습한다. 이 과정을 $d$번 반복한다.

**3단계: Re-training**
감지된 이상에 대해 재구성 오류가 높은 것만 선택적으로 학습하여 정상과 이상의 분리도를 높인다. 임계값 $T_{retrain}$을 초과하는 이상 후보만 음의 가중치 $w_i = w_{anomaly} < 0$을 부여한다.

가중치가 적용된 재구성 손실 함수는 다음과 같다.

$$L_{\mathbf{w}} = \sum_{i=1}^{N} w_i L(\mathbf{x}\_i, g(f(\mathbf{x}\_i)))$$

## 4. 실험 및 결과

### 데이터셋

MNIST와 Fashion-MNIST에서 실험이 수행되었다. MNIST에서는 숫자 '0'을 정상, 숫자 '2'를 이상으로 설정했고, Fashion-MNIST에서는 'T-shirt'를 정상으로 설정하고 'Boot' 또는 'Pullover'를 이상으로 설정했다. 학습 데이터의 이상 비율 $\alpha$는 5%, 1%, 0.1%로 다양하게 설정했다.

### 평가 지표

Balanced Accuracy (BAcc)를 주요 평가 지표로 사용했으며, 이는 TPR과 TNR의 평균으로 정의된다.

$$\text{BAcc} = \frac{\text{TPR} + \text{TNR}}{2} \in [0, 1]$$

### 주요 결과

MNIST에서 5% 이상 비율로 학습할 때, ITSR은 기존 AE 대비 30% 이상의 BAcc 향상을 달성했다. AAE 대비에서도 20% 이상의 향상을 보였다. Fashion-MNIST의最难한 경우('T-shirt' vs 'Pullover')에서 AAE 대비 30%의 성능 향상이 있었다.

ITSR은 학습 중 관찰된 유형의 이상에 대해 특히 강건한 성능을 보였다. 학습 데이터에 0.1%의 이상만 포함되어 있어도 효과적으로 동작하며, 관찰되지 않은 유형의 이상에 대해서도 기존 방법 대비 동등하거나 약간 더 나은 성능을 유지했다.

### 재구성 오류 분석

ITSR 적용 전후를 비교하면, 정상 클래스의 재구성 오류는 거의 변화가 없는 반면 이상 클래스의 재구성 오류가 크게 증가했다. 이는 ITSR이 정상 클래스를 강건하게 표현하도록 학습하면서 이상에 대한 민감도는 감소시키기 때문이다.

## 5. 강점, 한계

### 강점

ITSR은 학습 데이터의 오염에 대한 강건성을 크게 향상시킨다. 소수의 이상 데이터만으로도 효과적으로 동작하며, 전문가의 레이블링 없이 자동적으로 학습 데이터를 정제한다. 재구성 오류와 잠재 우도의 결합을 통해 단일 기준보다 더 나은 탐지 성능을 달성한다. 레이블 정보가 필요하지 않아 비지도 학습 설정에서 적용 가능하다.

### 한계 및 비판적 해석

1-class SVM의 파라미터 $\nu$와 재학습 임계값 설정이 데이터에 따라 다르며, 이를 올바르게 설정하기 위해서는 데이터에 대한 사전 지식이 필요하다. 실험이 MNIST와 Fashion-MNIST로 제한되어 있어 복잡한 실제 이미지나 다른 도메인에 대한 검증이 필요하다. 잠재 공간의 차원 수와 사전 분포 선택이 성능에 영향을 미칠 수 있으며, 이에 대한 체계적인 분석이 부족하다.

## 6. 결론

ITSR은 Adversarial Autoencoder의 잠재 공간 특성을 활용하여, 학습 데이터에 포함된 이상을 반복적으로 식별하고 제외하는 새로운 이상 탐지 방법을 제안했다. 기존 최첨단 방법 대비 향상된 성능을 보이며, 깨끗한 학습 데이터에 대한 의존성을 줄였다.

이 연구는 품질 검사, 의료 영상 분석 등 낮은 이상 비율이 예상되는 실제 응용 분야에서 유용하게 적용될 수 있다. 레이블 없이도 자동으로 학습 데이터를 정제할 수 있어 대규모 데이터셋에서 전문가 개입을 줄일 수 있다. 향후 연구에서는 다른 고차원 데이터 유형(시계열, 스펙트로그램 등)으로의 확장, 준지도 학습 또는 능동 학습과의 결합, 다양한 사전 분포 선택의 영향 분석等方面의 발전이 기대된다.
