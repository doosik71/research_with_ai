# Group Anomaly Detection using Deep Generative Models

- **저자**: Raghavendra Chalapathy, Edward Toth, Sanjay Chawla
- **발표연도**: 2018
- **arXiv**: https://arxiv.org/abs/1804.04876

## 1. 논문 개요

본 논문의 목표는 개별 데이터 포인트의 점 이상(point anomaly)이 아닌, 데이터 포인트의 모음(group)으로 구성된 이상을 탐지하는 것이다. 특히 이미지의 픽셀이나 시각적 특징의 불규칙한 혼합과 같은 불규칙한 그룹 분포(irregular group distributions)에 중점을 두고 그룹 이상 탐지(group anomaly detection, GAD)를 수행한다.

연구의 핵심 문제는 기존의 점 이상 탐지 방법들이 이미지나 복잡한 그룹 구조에서 그룹 차원의 이상을 효과적으로 탐지하지 못한다는 점이다. 예를 들어, 고에너지 물리학에서 Higgs 보손은 개별 충돌 이벤트가 아닌 충돌 이벤트의 모음에서 slight excess로 관찰되는데, 점 이상 탐지 방법들은 이러한 이상 행동을 구분할 수 없다.

연구의 중요성은 고에너지 입자 물리학, 소셜 미디어, 의료 영상 등 다양한 실제 응용 분야에서 비정상적이고 이상한 현상을 탐지하는 데 필수적이기 때문이다.

## 2. 핵심 아이디어

논문의 중심 아이디어는 그룹 참조 함수(group reference function)를 사용하여 그룹 이상을 탐지하는 딥 제너레이티브 모델(deep generative models, DGM)을 제안하는 것이다. 이미지를 픽셀이나 시각적 특징(예: 고양이 이미지에서 수염)의 그룹으로 모델링하고, 이 그룹들이 정상적인 패턴에서 얼마나 벗어나는지 측정한다.

기존 접근 방식과의 주요 차별점은 다음과 같다. 첫째, Mixture of Gaussian Mixture Models(MGMM), One-Class Support Measure Machines(OCSMM), One-Class Support Vector Machines(OCSVM) 등의 기존 GAD 방법은 특징 추출과 전처리에 크게 의존하지만, DGM은 엔드투엔드 방식으로 그룹 표현을 학습한다. 둘째, DGM은 더 큰 규모의 그룹 관찰을 통해 적절한 모델을 학습할 수 있으며, unseen 데이터에 대한 일반화 능력(inductive)을 가진다. 셋째, AAE는 KL divergence 대신 adversarial learning을 사용하여 더 넓은 범위의 분포를 prior로 학습할 수 있다.

## 3. 상세 방법 설명

### 3.1 문제 정의

그룹들의 집합 $\mathcal{G} = \{{\bf G}\_m\}\_{m=1}^{M}$이 주어졌을 때, $m$번째 그룹은 $N_m$개의 관찰을 포함하며 ${\bf G}\_m = (X_{ij}) \in \mathbb{R}^{N_m \times V}$로 표현된다. 그룹 행동은 특성화 함수 $f: \mathbb{R}^{N_m \times V} \to \mathbb{R}^{D}$로 포착되고, 집계 함수 $g: \mathbb{R}^{M \times D} \to \mathbb{R}^{D}$로 결합된다. 그룹 참조는 다음과 같이 정의된다:

$$\mathcal{G}^{(ref)} = g\Big[\{f({\bf G}\_m)\}\_{m=1}^{M}\Big]$$

거리 함수 $d(\cdot, \cdot) \geq 0$를 사용하여 특정 그룹이 그룹 참조로부터의偏离를 측정한다.

### 3.2 Autoencoder 기반 접근

Autoencoder는 인코더 $f_\phi$를 통해 입력을 잠재 표현으로 임베딩하고, 디코더 $g_\psi$가 잠재 표현에서 입력을 재구성한다. 재구성 손실은 다음과 같이 정의된다:

$$L_r(G_m, \hat{G}\_m) = \|G_m - \hat{G}\_m\|^2$$

Autoencoder는 재구성 오차를 이상 점수로 활용하여, 재구성 오차가 큰 데이터 포인트를 이상으로 간주한다.

### 3.3 Variational Autoencoder (VAE)

VAE는 잠재 변수 $z$를 추론할 때 제약 조건을 부과한다. 인코더가 생성한 잠재 코드는 사전 분포 $P(G_m)$를 따르도록 제한된다. 손실 함수는 다음과 같다:

$$\mathcal{L}(G_m, \hat{G}\_m) = L_r(G_m, \hat{G}\_m) + KL(f_\phi(z|x) \| g_\psi(z))$$

VAE는 재구성 확률 또는 재구성 오차를 사용하여 이상 점수를 계산한다.

### 3.4 Adversarial Autoencoder (AAE)

VAE의 주요 제한 사항은 KL divergence 적분에 대한 닫힌 형식 해석적 해가 대부분의 분포에서 불가능하다는 것이다. AAE는 adversarial learning을 채택하여 잠재 코드의 prior로 더 넓은 분포 집합을 학습한다.

AAE의 손실 함수는 다음과 같다:

$$L_G = \frac{1}{M'} \sum_{m=1}^{M'} \log D(z_m) \quad \text{and} \quad L_D = -\frac{1}{M'} \sum_{m=1}^{M'} \big[\log D(z'_m) + \log(1 - D(z_m))\big]$$

여기서 $z$는 인코더가 생성한 잠재 코드이고, $z'$은 실제 prior $P(z)$에서 샘플링된 것이다.

### 3.5 그룹 이상 탐지 알고리즘

AAE와 VAE를 학습한 후, 그룹 참조 $\mathcal{G}^{(ref)}$를 얻는다. 각 그룹에 대해 $\mathcal{G}^{(ref)}$로부터의 거리 $s_m = d(\mathcal{G}^{(ref)}, {\bf G}\_m)$를 계산하고, 점수를 내림차순으로 정렬한다. $\mathcal{G}^{(ref)}$에서 가장 먼 그룹이 더 이상으로 간주된다.

## 4. 실험 및 결과

### 4.1 합성 데이터: 회전된 가우시안 분포

M=500개의 정상 그룹(상관관계 $\rho=0.7$)과 50개의 이상 그룹(상관관계 $\rho=-0.7$)을 생성하여 실험하였다. 결과는 다음과 같다:

| 방법  | M=550 |       | M=5050 |       |
|:-----:|:-----:|:-----:|:------:|:-----:|
|       | AUPRC | AUROC | AUPRC  | AUROC |
| AAE   | 0.906 | 0.500 | 1.000  | 1.000 |
| VAE   | 0.900 | 0.500 | 1.000  | 1.000 |
| MGM   | 0.978 | 0.818 | 0.998  | 0.822 |
| OCSMM | 0.943 | 0.610 | 0.994  | 0.630 |
| OCSVM | 0.921 | 0.501 | 0.990  | 0.531 |

DGM은 더 큰 학습 세트(M=5050)에서 최고 성능을 달성하였으며, 이는 DGM이 적절한 모델을 학습하기 위해 더 많은 그룹 관찰을 필요로함을 보여준다.

### 4.2 고양이 이미지에서 호랑이 탐지

5000장의 고양이 이미지와 50장의 호랑이 이미지를 결합하여 점 기반 그룹 이상(또는 이미지 이상) 탐지를 실험하였다. 호랑이 이미지의 줄기와 같은 개별 특징이 점 이상으로 작용한다.

### 4.3 회전된 개체 탐지

5000장의 고양이 이미지와 50장의 회전된 고양이 이미지를 사용하여 분포 기반 그룹 이상 탐지를 실험하였다. 회전된 고양이 이미지는 정상 고양이 이미지와 비교하여 비정상적인 분포를 가진다.

### 4.4 고양이와 개 탐지

2500장의 고양이, 2500장의 개, 그리고 50장의 고양이와 개가 섞인 이미지를 포함하는 데이터셋으로 분포 기반 그룹 이상 탐지를 수행하였다.

### 4.5 연결된 장면 이미지 탐지

"도심", "산", "해안" 카테고리에서 각각 100장의 이미지와 66장의 두 가지 장면 카테고리가 연결된 이상 이미지로 실험하였다.

|:-----:|:------:|:-------------:|:---------:|:-----:|

AAE는 장면 데이터를 제외한 대부분의 실험에서 최고의 탐지 성능을 달성하였다.

## 5. 강점, 한계

### 5.1 강점

본 논문의 강점은 다음과 같다. DGM은 End-to-end 방식으로 그룹 표현을 학습하여 수동 특징 추출에 대한 의존성을 줄인다. AAE는 adversarial learning을 통해 더 넓은 범위의 분포를 prior로 학습할 수 있다. 더 큰 학습 데이터셋에서 기존 GAD 방법보다 우수한 성능을 보인다. 모델이 unseen 데이터에 일반화될 수 있는 inductive 특성을 가진다.

### 5.2 한계 및 미해결 질문

본 연구의 한계는 다음과 같다. DGM은 적절한 모델을 학습하기 위해 더 많은 그룹 관찰을 필요로 한다. 소규모 그룹 데이터셋(M=366)에서는 기존 GAD 방법보다 성능이 낮다. 계산 시간이 MGM이나 OCSVM보다 길다. 장면 데이터셋과 같이 시각적 특징이 로컬에서 정상이지만 전역적으로 비정상적인 경우에 대한 탐지가 어렵다.

## 6. 결론

본 논문의 주요 기여는 그룹 이상 탐지를 위한 딥 제너레이티브 모델(AAE와 VAE)을 처음으로 공식화하고 적용한 것이다. DGM은 합성 및 실제 이미지 데이터셋에서 기존 최첨단 GAD 기법을 능가하는 성능을 보여주었다.

향후 연구 방향으로 시계열 데이터의 시간적 변화를 탐지하기 위한 순환 신경망(RNN) 활용, 커널 방법의 빠른 근사를 통한 OCSMM 복잡성 개선, 그리고 더 복잡한 그룹 구조에 대한 확장이 있다. 이 연구는 고에너지 물리학, 소셜 미디어 분석, 의료 영상 등 다양한 영역에서 그룹 이상 탐지 응용의 가능성을 열어주었다.
