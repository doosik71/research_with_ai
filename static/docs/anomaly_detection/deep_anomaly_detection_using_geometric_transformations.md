# Deep Anomaly Detection Using Geometric Transformations

- **저자**: Izhak Golan, Ran El-Yaniv
- **발표연도**: 2018
- **arXiv**: https://arxiv.org/abs/1805.10917

## 1. 논문 개요

### 논문의 목표

본 논문의 목표는 이미지의 이상치 탐지(anomaly detection) 문제를 해결하기 위한 새로운 기법을 제안하는 것이다. 특히 "normal" 클래스(예: 개의 이미지)로만 구성된 샘플이 주어졌을 때, 분포 외(out-of-distribution) 이미지(예: 개가 아닌 객체)를 탐지할 수 있는 딥 뉴럴 모델을 훈련시키는 것을 목적으로 한다.

### 연구 문제

기본적인 이상치 탐지 문제에서는 "정상" 클래스의 인스턴스가 포함된 샘플이 주어지며, 분포 외 "비정상" 인스턴스를 탐지할 수 있는 분류기를 구성하는 것이 목표이다. 기존 방법들은 재구성 기반(오토인코더, GAN) 또는 저밀도 추정 방법(One-Class SVM, KDE)을 사용하나, 고차원 이미지 데이터에서 성능 저하 문제가 있다.

### 문제의 중요성

자율주행 자동차나 가정용 로봇과 같은 미래 머신러닝 응용 분야에서 분포 발산이나 노이즈 관찰로 인한 통계적 불일치에 노출될 수 있다. 이러한 미션 크리티컬 응용에서 이상치 탐지의 해결은 매우 중요하다. 특히 딥 뉴럴 모델의 맥락에서 이상치 탐지 문제에 대한 결론적 이해가 부족한 상황에서, CIFAR-100과 같은 대규모 데이터셋에서의 연구가 필요하다.

## 2. 핵심 아이디어

### 중심 직관

본 논문의 핵심 아이디어는 정상 이미지에 다양한 기하학적 변환을 적용하고, 이를 분류하도록 다중 클래스 모델을 훈련시키는 것이다. 변환을 구별하도록 훈련함으로써, 모델은 정상 클래스에 고유한 기하학적 피처를 학습하게 된다. 테스트 시에는 변환된 이미지에서 softmax 활성화의 분포를 분석하여 이상치를 탐지한다.

### 기존 접근 방식과의 차별점

기존 방법들은 오토인코더나 GAN의 재구성 오차에 의존하거나, 저밀도 추정 방법(One-Class SVM, KDE)을 사용한다. 그러나 본 방법은 재구성 모듈을 완전히 배제하고, 판별적(discriminative) 방식으로 기하학적 변환을 분류하도록 훈련한다. 이 접근법은 생성 모델의 불안정한 훈련 문제를 피하면서도 효과적인 이상치 탐지가 가능하다.

## 3. 상세 방법 설명

### 자기 레이블 데이터셋 생성

변환의 집합 $\mathcal{T} = \{T_0, T_1, ..., T_{k-1}\}$가 주어졌을 때, 여기서 $T_0(x) = x$는 항등 변환이다. 자기 레이블 데이터셋 $S_{\mathcal{T}}$는 다음과 같이 정의된다:

$$S_{\mathcal{T}} \triangleq \{(T_j(x), j) : x \in S, T_j \in \mathcal{T}\}$$

즉, 각 변환된 이미지에 적용된 변환의 인덱스를 레이블로 붙인다. 이 과정은 $|T| \cdot |S|$ 크기의 자동 레이블 다중 클래스 데이터셋을 생성한다.

### 분류기 훈련

생성된 데이터셋을 사용하여 표준 교차 엔트로피 손실 함수로 $k$ 클래스 이미지 분류 모델 $f_\theta$를 훈련시킨다. 사용된 네트워크 아키텍처는 Wide Residual Network (WRN)이다.

### Dirichlet Normality Score

테스트 시, 주어진 이미지 $x$에 각 변환을 적용하고 분류기를 통과시켜 $|T|$ 개의 softmax 응답 벡터를 얻는다. 정규성 점수는 다음과 같이 정의된다:

$$n_S(x) \triangleq \sum_{i=0}^{k-1} \log p(\mathbf{y}(T_i(x)) | T_i)$$

여기서 $\mathbf{y}(x) \triangleq \text{softmax}(f_\theta(x))$는 분류기의 softmax 응답 벡터이다.

각 조건부 분포를 Dirichlet 분포로 근사한다: $\mathbf{y}(T_i(x)) | T_i \sim \text{Dir}(\boldsymbol{\alpha}\_i)$

최종 정규성 점수는 다음과 같이 단순화된다:

$$n_S(x) = \sum_{i=0}^{k-1} (\tilde{\boldsymbol{\alpha}}\_i - 1) \cdot \log \mathbf{y}(T_i(x))$$

### 단순화된 정규성 점수

실험 초기에 다음의 단순화된 정규성 점수도 사용되었다:

$$\hat{n}\_S(x) \triangleq \frac{1}{k} \sum_{j=0}^{k-1} [\mathbf{y}(T_j(x))]_j$$

이 점수 함수는 Dirichlet 파라미터 추정이 필요 없어 구현이 간단하며, Dirichlet 점수보다 약간 낮은 성능을 보이지만 여전히 우수한 결과를 달성한다.

## 4. 실험 및 결과

### 데이터셋

네 가지 이미지 데이터셋에서 실험을 수행하였다. CIFAR-10(32×32 컬러 이미지, 10개 클래스), CIFAR-100(32×32 컬러 이미지, 100개 클래스), Fashion-MNIST(28×28 회색조 이미지, 10개 카테고리), 그리고 CatsVsDogs(64×64 이미지)가 사용되었다.

### 기준선 방법

One-Class SVM(RAW-OC-SVM, CAE-OC-SVM), End-to-End One-Class Deep SVDD(E2E-OC-SVM), Deep Structured Energy-Based Model(DSEBM), Deep Autoencoding Gaussian Mixture Model(DAGMM), 그리고 ADGAN과 비교하였다.

### 주요 정량적 결과

CIFAR-10에서 제안된 방법은 평균 AUROC 86.0%를 달성하여, 최고 기준선(CAE-OC-SVM의 64.8%)보다 32% 향상되었다. CatsVsDogs 데이터셋에서는 최고 기준선(DSEBM의 56.1%)보다 67% 향상된 88.8%의 AUROC를 달성하였다. Fashion-MNIST에서는 93.5%의 AUROC를 달성하여 경쟁력 있는 성능을 보였다.

### 실험적 직관 검증

MNIST 데이터셋에서 세 가지 실험을 통해 직관을 검증하였다. '8'을 정상으로, '3'을 이상으로 설정하고 수평 뒤집기 변환만 사용한 실험에서, '8'이 수평 뒤집기에 불변(imvariant)하기 때문에 분류기가 구별 피처를 학습하기 어렵고 AUROC 0.646을 기록했다. 반대로 '3'을 정상으로 설정하면 AUROC 0.957을 달성하였다. 이는 변환이 정상 클래스의 특성을 얼마나 잘 보존하는지가 성능에 중요한 영향을 미친다는 것을 보여준다.

### 분포 외 샘플 식별

라벨이 있는 다중 클래스 데이터셋에서 분포 외 샘플을 식별하는 문제에서도 실험을 수행하였다. CIFAR-10으로 훈련하고 tiny-imagenet을 이상으로 설정한 실험에서, ODIN 방법(AUROC 92.1/AUPR-In 89.0/AUPR-Out 93.6)을 능가하여 95.7/96.1/95.4를 달성하였다.

## 5. 강점, 한계

### 강점

본 논문의 주요 강점은 다음과 같다. 첫째, 재구성 기반 방법(오토인코더, GAN)을 완전히 배제하고 판별적 방식으로 이상치 탐지를 수행한다. 이는 생성 모델의 불안정한 훈련 문제를 해결한다. 둘째, 다중 클래스 분류기로서 구현이 간단하며, 표준 분류 task와 동일한 난이도로 구현 가능하다. 셋째, 다양한 이미지 데이터셋에서 기존 최첨단 방법들보다 현저히 우수한 성능을 보였다. 특히 CIFAR-10에서 32%, CatsVsDogs에서 67%의 AUROC 향상을 달성하였다. 넷째, WRN 아키텍처를 활용하여 최첨단 분류 성능을 활용한다.

### 한계 및 미해결 질문

본 논문에는 몇 가지 한계가 존재한다. 첫째, 기하학적 변환의 선택이 성능에 중요한 영향을 미치며, 변환 선택에 대한 이론적 근거가 부족하다. 둘째, CIFAR-100과 같이 정상 클래스가 다양한 하위 클래스를 포함하는 경우 성능이 저하된다. 이는 정상 클래스의 다양성이 증가할 때 구별 피처 학습이 어렵기 때문이다. 셋째, 사용된 72개의 기하학적 변환에 대한详细介绍가 제한적이다. 넷째, 원본 이미지보다 작은 이미지(예: MNIST)에서 실험 결과가 제시되지 않았다.

## 6. 결론

본 논문은 이미지의 이상치 탐지를 위한 새로운 방법을 제안하였다. 정상 이미지에 다양한 기하학적 변환을 적용하고 이를 분류하도록 다중 클래스 분류기를 훈련시키는 판별적 접근법을 사용한다. 테스트 시에는 변환된 이미지에서 softmax 응답 벡터의 분포를 분석하여 정규성 점수를 계산한다.

CIFAR-10, CIFAR-100, Fashion-MNIST, CatsVsDogs 등 다양한 데이터셋에서의 실험 결과, 제안된 방법이 기존 최첨단 이상치 탐지 방법들을 크게 능가함을 보여주었다. 특히 CIFAR-10에서 32%, CatsVsDogs에서 67%의 AUROC 향상을 달성하여, 판별적 접근법이 이미지 이상치 탐지에서 매우 효과적임을 입증하였다.

향후 연구 방향으로는 기하학적 변환 사용에 대한 이론적 기반 개발, 주어진 훈련 데이터셋에 최적화된 변환 선택 방법, 그리고 비전以外の 응용 분야(열린 세계 시나리오, 다중 클래스 분류 성능 개선 등)로의 확장이 있다.
