## 연구 배경 및 문제 정의

그래프는 개체 간 관계와 상호 의존성을 표현할 수 있어 네트워크 보안, 금융 사기, 소셜 네트워크, IoT 등 다양한 도메인에서 이상 탐지에 활용된다. 그러나 그래프 구조의 복잡성과 동적 변화는 전통적인 이상 탐지 방법으로 처리하기 어렵다. 기존 서베이들은 일반적 이상 탐지 또는 일부 그래프 기반 기법을 다루었으나, 그래프 구조 데이터 자체를 중심에 둔 포괄적 분류와 비교가 부족했다. 본 논문은 그래프 구조 데이터에서의 이상 탐지 연구를 체계적으로 정리하기 위해, 이상 구성 요소, 그래프 유형, 탐지 기법, 이상 유형, 응용 분야의 다섯 관점으로 분류 체계를 제시한다.

## 핵심 기여

- 그래프 구조 데이터에서의 이상 탐지 연구를 다섯 축(이상 구성 요소, 그래프 유형, 탐지 기법, 이상 유형, 응용 분야)으로 정리한 통합적 분류 체계 제시.
- 정적/동적/스트리밍 그래프 환경에서의 이상 탐지 문제를 구분하고, 각 환경의 난제를 정리.
- 확률·통계, 행렬/텐서 분해, 거리·유사도, 그래프 커널, 그래프 신경망 등 기법별 장단점과 적용 맥락을 비교.
- 그래프 기반 이상 탐지의 핵심 도전 과제(스케일, 시간성, 특징 공간 확장, 원패스 처리 등)와 향후 연구 방향 제안.

## 방법론 요약

### 1. 이상 구성 요소 기반 분류

- 이상 노드: 연결 패턴, 커뮤니티 이동, 라벨 분포의 비정상성으로 정의.
- 이상 엣지: 엣지 가중치의 비정상적 증가, 시점 간 변화의 급격성으로 정의.
- 이상 (서브)그래프: 서브그래프 구조의 비정상성, 시간에 따른 구조 변화로 정의.

### 2. 그래프 유형 기반 분류

- 정적 그래프: 단일 스냅샷 기반 분석. 오프라인 탐지가 주류.
- 동적 그래프: 시간 변화가 포함된 그래프. 스트리밍 그래프(스냅샷 집계)와 스트리밍 엣지(실시간 처리)로 세분.
- 동적 그래프에서는 데이터 규모 증가, 특징 공간 확장, 원패스 처리 제약이 핵심 난제.

### 3. 그래프 기반 탐지 기법 분류

- 확률/통계 기반: 정상 패턴 분포를 모델링하고 편차를 이상으로 정의.
- 행렬/텐서 분해 기반: 저차원 구조 복원 오차를 이용해 이상을 탐지.
- 거리/유사도 기반: 커뮤니티, 서브그래프 구조 간 거리 차이를 이용.
- 그래프 커널/스케치 기반: 그래프 구조 특성을 요약하여 이상 판별.
- 그래프 신경망 기반: 표현 학습을 통해 이상 노드, 링크, 그래프를 탐지.

### 4. 이상 유형 분류

- 희소 이상(outliers): 소수 노드나 엣지의 비정상적 편차.
- 집단 이상(burst/group): 특정 시점에 그룹 단위로 급증하는 이상.
- 급격 이상(sudden): 특정 시점에 구조가 급변.
- 점진 이상(gradual): 구조 변화가 누적되며 발생.

### 5. 응용 분야 정리

- 네트워크 보안, 금융 사기 탐지, 소셜 네트워크 스팸/봇 탐지, IoT 이상 탐지, 추천 시스템 부정 행위 탐지 등.
- 응용마다 이상 구성 요소와 그래프 유형이 달라 기법 선택이 달라짐.

## 실험 설정과 결과

본 논문은 실험을 직접 수행하지 않고, 기존 연구들을 비교·정리한 서베이다. 따라서 통일된 실험 설정이나 단일 벤치마크 결과를 제공하지 않는다. 대신 다양한 기법들의 장단점, 적용 분야, 확장성 및 실시간 처리 가능성 등을 비교하여 종합적인 이해를 제공한다.

## 한계 및 향후 연구 가능성

- 공통 벤치마크 부재로 인해 방법 간 정량 비교가 어렵다.
- 대규모 그래프와 스트리밍 환경에서 실시간 탐지가 여전히 어렵다.
- 동적 그래프에서는 시간적 패턴과 구조 변화를 동시에 반영하는 모델이 부족하다.
- GNN 기반 접근은 효과적이지만 계산 비용과 해석 가능성 문제가 존재.

향후 연구는 대규모·실시간 환경에서의 효율적 탐지, 그래프 기반 이상 설명 가능성 강화, 멀티모달 그래프 데이터 통합, 그리고 모델-불변 탐지 기법 개발로 확장될 필요가 있다.

## 실무적 또는 연구적 인사이트

- 그래프 기반 이상 탐지는 이상 구성 요소(노드/엣지/서브그래프)와 그래프 유형(정적/동적/스트리밍)에 따라 접근 방법이 달라진다.
- 스트리밍 엣지 환경에서는 원패스 제약과 계산 복잡도가 가장 큰 장애로, 스케치/요약 기반 기법이 유용하다.
- GNN 기반 방법은 표현 학습 측면에서 강력하지만, 적용 시 비용과 해석 가능성 문제를 함께 고려해야 한다.
- 실무에서는 이상 유형(희소/집단/급격/점진)을 명시한 후 탐지 기법을 선택하는 것이 효과적이다.
