# Anomaly Detection for a Water Treatment System Using Unsupervised Machine Learning

- **저자**: Jun Inoue, Yoriyuki Yamagata, Yuqi Chen, Christopher M. Poskitt, Jun Sun
- **발표연도**: 2017
- **arXiv**: https://arxiv.org/abs/1709.05342

## 1. 논문 개요

본 논문의 목표는 Cyber-Physical System(CPS)에 대한 이상 감지에 비지도 기계학습(unsupervised machine learning)을 적용하는 것이다. 특히 정유수처리 시스템(Secure Water Treatment, SWaT) 테스트베드를 대상으로 하여, 정상 운영 조건에서 생성된 로그 데이터로 학습된 모델을 기반으로 이상 동작을 탐지하는 두 가지 방법을 제안하고 비교한다.

연구 문제는 CPS의 복잡성으로 인해 정확한 수학적 모델을 구축하는 것이 실무적으로 매우 어렵다는 점이다. 기존 모델 기반 접근법은 시스템의 구성에 대한 사전 지식과 운영 로그가 모두 필요하며, 규칙 기반 접근법은 정확한 규칙을 도출하는 것이 현실적으로 어렵다. 따라서 비지도 기계학습을 활용하면 시스템에 대한 이해 없이도 데이터 로그만으로 모델을 구축할 수 있다는 것이 연구의 핵심 동기이다.

이 연구의 중요한점은 실제 인프라를 대상으로 한 보안 공격 시뮬레이션 데이터를 통해 방법론의 유효성을 검증한다는 것이다.

## 2. 핵심 아이디어

논문의 중심 아이디어는 두 가지 비지도 기계학습 방법을 적용하여 CPS의 시계열 데이터에서 이상을 탐지하는 것이다. 첫 번째 방법은 Long Short-Term Memory(LSTM) 아키텍처로 구성되고 feed-forward 레이어가 이어지는 Deep Neural Network(DNN)를 활용한 확률적 이상치 탐지(probabilistic outlier detection)이다. 두 번째 방법은 one-class Support Vector Machine(SVM)을 활용한 시계열 데이터에서 비정상적인 패턴을 찾는 접근법이다.

기존 접근 방식과의 주요 차별점은 다음과 같다. 기존 예측 오류 기반 방법(Malhotra et al.)은 시스템 상태에 따라 정상 데이터의 예측 오류가 변동할 수 있어 균일한 임계값 사용이 어렵고 recall이 약 10%에 불과하지만, 본 논문의 DNN은 모든 시간 단계에 대해 직접적으로 확률 분포를 계산하므로 이 문제를回避한다. 또한 이산값 데이터와 연속값 데이터의 혼합을 동시에 처리할 수 있다는 점에서 에너지 기반 모델(Zhai et al.)과 달리 확장 가능성이 있다.

## 3. 상세 방법 설명

### 3.1 SWaT 테스트베드 개요

SWaT는 싱가포르 기술디자인 대학에 구축된 축소된 규모이지만 완전 운영 가능한 정수 시설로, 분당 5갤런의 음용수를 생산할 수 있다. 전체 시스템은 6단계로 구성되며, 각 단계는 고유한 PLC(Programmable Logic Controller)에 의해 제어된다. 시스템에는 25개의 센서와 26개의 액추에이터가 포함되어 있다.

데이터셋은 11일 연속 운영 중 수집되었으며, 7일은 정상 조건, 4일은 36가지 공격 시나리오가 적용되었다. 공격은 네트워크 통신 링크를 통해 구현되어 PLC, 펌프, 밸브에 도달하기 전에 센서 데이터와 액추에이터 신호를 조작한다.

### 3.2 DNN 기반 이상 감지 방법

DNN 기반 방법은 확률적 이상치 탐지를 구현한다. 핵심은 모든 시간 단계에서 다음 상태의 확률 분포를 직접 계산하는 것이다. $i$번째 로그 항목 $l_i \equiv \langle \bar{a}(i), \bar{s}(i) \rangle$가 주어졌을 때, outlier factor는 다음과 같이 정의된다:

$$-\log q(l_i \mid \mathbf{l}\_{i-1})$$

실제로 직접 계산할 수 없으므로 다음과 같이 분해한다:

$$-\sum_{j=1}^{n}\log q_j(a_j(i) \mid \mathbf{l}\_{i-1}, a_1(i), \ldots, a_{j-1}(i)) - \sum_{k=1}^{m}\log r_k(s_k(i) \mid \mathbf{l}\_{i-1}, \bar{a}(i), s_1(i), \ldots, s_{k-1}(i))$$

여기서 $q_j$는 이산 분포(액추에이터 위치에 대해 softmax 함수 사용), $r_k$는 가우시안 분포로 근사화된다.

### 3.3 DNN 아키텍처

DNN 아키텍처는 다음과 같은 구조를 따른다. LSTM 레이어는 $i-1$번째 위치까지의 로그 항목을 입력으로 받아 은닉 상태를 출력한다. 이 출력은 먼저 softmax 함수를 사용하여 각 액추에이터 위치의 확률을 예측하는 데 사용된다. 출력 레이어는 $i$번째 로그 항목의 실제 액추에이터 위치를 가져와서 쌍선형 함수(bilinear function)를 통해 은닉층 값과 혼합한다. 쌍선형 함수의 출력시 완전 접속층을 거쳐, 다음의 은닉층의 출력에 연결된다. 최종 출력은 첫 번째 센서 값의 예측 평균과 분산이다. 이 과정을 모든 센서 값에 대해 반복하고, 모든 액추에이터 위치와 센서 값의 로그 우도를 합산하여 outlier factor를 얻는다.

### 3.4 학습 및 비용 함수

학습을 위해 크로스 엔트로피(cross entropy)를 비용 함수로 사용한다:

$$C(p, q) = \int -p(z)\log q(z)d\mu$$

진짜 분포 $p$를 모르기 때문에 디랙 델타 함수 $\delta(z-x)$로 근사화하면:

$$C(p, q) = -\log q(x) = \sum_{i=1}^{N} -\log q(l_i \mid \mathbf{l}\_{i-1})$$

이는 모든 outlier factor의 합이며, 학습 데이터의 모든 outlier factor 합을 비용 함수로 사용한다.

### 3.5 One-class SVM 기반 이상 감지

One-class SVM은 슬라이딩 윈도우 방법을 사용하여 시계열 데이터를 개별 피처 벡터로 변환한다. $l_i$가 $i$번째 로그 항목이고 $w$가 윈도우 크기일 때, $W_i \equiv \langle l_i, l_{i+1}, \ldots, l_{i+w-1} \rangle$ 형태의 튜플을 윈도우라 한다. 학습 단계에서는 학습 데이터에서 모든 윈도우를 추출하여 one-class SVM 알고리즘에 공급한다. 테스트 단계에서는 각 윈도우에 대해 정상/비정상으로 분류하고, 윈도우 내 하나 이상의 로그 항목이 비정상으로 레이블되면 해당 윈도우를 비정상으로 분류한다.

### 3.6 데이터 정규화

SVM의 경우 학습 데이터와 테스트 데이터를 각각 자체 평균과 분산으로 정규화하는 것이 효과적이었다(F measures가 약 20-30%에서 80%까지 향상). 반면 DNN은 학습 데이터의 평균과 분산을 사용하여 테스트 데이터를 정규화할 때 더 나은 성능을 보였다.

## 4. 실험 및 결과

### 4.1 실험 설정

DNN은 Chainer 딥러닝 프레임워크로 구현되었으며, SVM은 scikit-learn 라이브러리(libsvm 백엔드)로 구현되었다. 학습에는 7일간의 정상 로그 데이터가 사용되었고, 평가에는 4일간의 공격 로그 데이터(36개 공격 시나리오)가 사용되었다.

하이퍼파라미터 튜닝 결과, DNN은 100차원의 중간 레이어와 58 에포크 학습이 최적이었다. SVM은 $w=4$, $\gamma=0.000818$, $\nu=0.00458$가 최적 파라미터로 도출되었다.

### 4.2 주요 정량적 결과

비교 결과, DNN이 SVM보다 전반적으로 더 나은 성능을 보였다:

| 방법          | Precision | Recall  | F measure |
| ------------- | --------- | ------- | --------- |
| DNN           | 0.98295   | 0.67847 | 0.80281   |
| One-class SVM | 0.92500   | 0.69901 | 0.79628   |

DNN은 더 높은 정밀도를 가지며(거짓 양성 적음), SVM은 약간 더 높은 재현율을 보인다. 그러나 전체적인 F measure는 DNN이 더 우수하다.

### 4.3 개별 공격별 탐지 성능

공격 유형별 분석 결과, 다음과 같은 경향이 관찰되었다. DNN은 센서 값이 일정한 비정상 값으로 변경되는 공격을 효과적으로 탐지하는 경향이 있다(예: Attack #10의 FIT-401을 <<0.7으로 설정, 탐지율 1.0). 반면 DNN은 점진적인 센서 값 변화나 비정상적인 액추에이터 움직임이 있는 공격을 탐지하는 데 어려움을 겪는다. SVM은 범위 밖 값을 탐지하는 데 더 효과적일 수 있지만 항상 그런 것은 아니다.

### 4.4 거짓 양성 분석

SVM은 센서 값의 급격한 변화에 대해 거짓 양성을 보고하는 경향이 있으며, 이는 고정된 슬라이딩 윈도우만 사용하기 때문에 장기적 추세가 전혀 고려되지 않기 때문이다. SVM은 전체 비공격 윈도우의 약 0.8%에서 거짓 알람을 발생시킨다.

## 5. 강점, 한계

### 5.1 강점

본 논문의 강점은 다음과 같다. 비지도 학습으로 시스템에 대한 사전 지식 없이도 모델을 구축할 수 있다. LSTM을 활용한 DNN은 비선형 역학을 가진 시스템의 특성을 명시적으로 가정하지 않고도 학습할 수 있다. 확률적 밀도 추정을 기반으로 하여 예측 오류 기반 방법보다 더 나은 성능을 달성한다. 실제 SWaT 테스트베드에서 36가지 실제 공격 시나리오를 통해 체계적으로 검증되었다.

### 5.2 한계 및 미해결 질문

본 연구의 한계는 다음과 같다. 두 방법 모두 점진적인 센서 값 변화 감지에 어려움을 겪는다. 비정상적인 액추에이터 동작 감지도 어려운 과제로 남아 있으며, 이를 극복하려면 제어기의 로직을 고려해야 할 수 있다. DNN의 학습 시간이 약 2주, 평가 시간이 8시간으로 SVM(각각 30분, 10분)보다 계산 비용이 훨씬 높다. 실험이 단일 테스트베드(SWaT)에서만 수행되었으므로 다른 CPS로의 일반화 가능성이 제한적이다. 또한 의도적으로 주입된 공격에 기반한 데이터이므로 실제 공격의 특성과는 다를 수 있다.

## 6. 결론

본 논문의 주요 기여는 CPS 이상 감지를 위한 두 가지 비지도 기계학습 방법(DNN과 one-class SVM)을 체계적으로 비교하고 평가한 것이다. SWaT 테스트베드에서의 실험 결과, DNN이 더 나은 F measure를 달성했으며 특히 거짓 양성 측면에서 SVM보다 우수한 성능을 보였다.

향후 연구 방향으로, 장기적 데이터 추세를 피처 벡터에 인코딩하여 성능을 개선하고, 개별 센서와 액추에이터에 대한 outlier factor를 계산하여 어느 구성 요소가 이상을 일으키는지 추론하는能力을 개발할计划이다. 또한 SWaT 시뮬레이터를 사용한 추가 실험과 LSTM-AE 등 다른 신경망 기반 방법과의 비교를 계획하고 있다.

실제 중요성 측면에서, 이러한 비지도 기계학습 기반 이상 감지 방법은 정수 시설뿐만 아니라 스마트 그리드, 철도 등 다양한 공공 인프라의 사이버 물리 시스템 보안에 기여할 수 있는 잠재력이 있다.
