# Denoising Architecture for Unsupervised Anomaly Detection in Time-Series

## 논문 메타데이터

- 제목: Denoising Architecture for Unsupervised Anomaly Detection in Time-Series
- 저자: Wadie Skaf, Tomáš Horváth
- 발표 연도: 2022
- arXiv ID: 2208.14337
- arXiv URL: https://arxiv.org/abs/2208.14337
- DOI: 10.1007/978-3-031-15743-1_17

## 연구 배경 및 문제 정의

시계열 이상 탐지는 금융, 보안, 의료 등 다양한 분야에서 핵심 과제지만, 이상 정의가 모호하고 레이블이 부족하며 시간적 상관관계가 복잡해 탐지가 어렵다. LSTM 기반 인코더-디코더(EncDec-AD)는 정상 시계열을 재구성하고 재구성 오차를 이상 점수로 활용하는 대표적 접근이지만, 전통적으로 정상 데이터만으로 학습하는 반지도 설정에 가깝다. 본 논문은 LSTM 오토인코더에 드롭아웃을 체계적으로 삽입한 “denoising architecture”를 제안하여, 레이블 없는 데이터(비지도) 환경에서도 성능과 학습 효율을 개선할 수 있음을 보여준다.

## 핵심 기여

- LSTM 오토인코더에 드롭아웃을 층별로 삽입한 denoising architecture 제안.
- Yahoo S5 데이터셋(실제 + 합성)에서 성능(Precision/Recall/F1)과 학습 속도(에폭 수) 개선을 실험적으로 제시.
- 데이터셋별 최적 드롭아웃 확률 _p_ 가 이상 비율과 상관됨을 관찰하고, 실무적 선택 기준을 제안.

## 방법론 요약

### 1. LSTM 오토인코더 기반 이상 탐지

- 시계열 입력을 인코더가 잠재 벡터로 압축, 디코더가 재구성.
- 재구성 오차를 이상 점수로 사용하고, 임계값 초과 시 이상으로 판정.

### 2. Denoising Architecture

- 각 LSTM 층 뒤에 드롭아웃을 삽입.
- 학습 중 일부 출력을 랜덤하게 0으로 만들어 모델이 정상 패턴에 더 강건하도록 유도.
- 결과적으로 정상 데이터 재구성은 유지하면서 이상 데이터 재구성 오차를 더 크게 만들도록 설계.

### 3. 핵심 설계 요지

- 드롭아웃 확률 _p_ 는 데이터의 이상 비율과 반비례 경향을 보이며, 이상이 많을수록 낮은 _p_ 가 효과적.
- 동일한 기본 LSTM 오토인코더에 대해 _p_ 를 변화시키며 성능 비교.

## 실험 설정과 결과

- 데이터셋: Yahoo S5 (A1 실제, A3/A4 합성, A2는 이상 없음으로 제외).
- 전처리: 각 시계열을 [-1, 1] 정규화, 슬라이딩 윈도 크기 24, 스텝 1.
- 평가 지표: Precision, Recall, F1-Score, 학습 에폭 수.

주요 결과는 다음과 같다.

- 실제 데이터(A1)에서 denoising architecture가 Precision/Recall/F1을 개선.
- 합성 데이터(A3, A4)에서는 성능 향상이 제한적이지만 학습 속도(에폭 수)가 개선.
- 최적 드롭아웃 확률은 A1에서 0.4, A4에서 0.2, A3에서는 아키텍처에 따라 0.1~0.2가 유효.

## 한계 및 향후 연구 가능성

- 개선 효과가 합성 데이터에서는 제한적이며, 실제 데이터에서 더 뚜렷한 개선이 관찰됨.
- 드롭아웃 확률 선택이 데이터 분포(이상 비율)에 민감하므로 자동 튜닝 전략이 필요.
- 다른 시계열 모델(Transformer 계열 등)과의 결합 또는 확장 가능성은 미검증.

## 실무적 또는 연구적 인사이트

- 비지도 이상 탐지에서 드롭아웃 기반 denoising은 단순하지만 효과적인 정규화 기법으로 작동할 수 있다.
- 이상 비율이 낮은 실제 환경에서 더 큰 성능 개선이 기대된다.
- 드롭아웃 확률은 이상 비율을 추정하거나 실험적으로 탐색해 결정하는 것이 현실적이다.
