# Scalable and Interpretable One-class SVMs with Deep Learning and Random Fourier Features

- **저자**: Minh-Nghia Nguyen, Ngo Anh Vien
- **발표연도**: 2018
- **arXiv**: https://arxiv.org/abs/1804.04888

## 1. 논문 개요

### 논문의 목표

본 논문의 목표는 대규모 고차원 데이터셋에서 이상 탐지(anomaly detection) 작업을 효과적으로 수행할 수 있는 새로운 모델을 제안하는 것이다. 특히, One-class Support Vector Machine (OC-SVM)과 딥러닝 기반 차원 감소 기법인 오토인코더(autoencoder)를 결합하여, 기존 접근법의 한계를 극복하고자 한다.

### 연구 문제

OC-SVM은 오랫동안 이상 탐지 분야에서 가장 효과적인 방법으로 널리 사용되어 왔다. 그러나 OC-SVM의 가장 큰 문제점은 대규모 및 고차원 데이터셋에서 작동할 때의 최적화 복잡성이다. 기존에는 매니폴드 학습이나 오토인코더와 같은 차원 감소 기법을 사용하여 이 문제를 완화하려고 했으나, 대부분의 기존 연구는 표현 학습(representation learning)과 이상 탐지를 별도로 수행하는 이단계(two-stage) 접근법을 사용하였다. 이러한 분리된 학습은 중요한 이상 탐지 정보가 잠재 공간에서 다르게 해석될 수 있어 최종 성능에 부정적인 영향을 미칠 수 있다.

### 문제의 중요성

이상 탐지는 네트워크 침입 탐지, 산업 제조 공정의 결함 탐지, 의료 분야의 질병 진단, 사이버 보안 등 광범위한 실용적 응용 분야에서 중요한 역할을 한다. 이러한 응용 분야에서는 대량의 고차원 데이터가 생성되며, 실시간 환경에서의 빠른 응답이 요구된다. 따라서 대규모 데이터에서도 효율적으로 작동하면서 해석 가능한 이상 탐지 모델의 개발은 중요한 연구 과제이다.

## 2. 핵심 아이디어

### 중심 직관 및 설계 아이디어

본 논문의 핵심 아이디어는 OC-SVM을 딥러닝 프레임워크에 통합하여 end-to-end로 학습할 수 있는 AE-1SVM (Autoencoder-based One-class SVM) 모델을 제안하는 것이다. 구체적으로 다음과 같은 세 가지 핵심 요소를 결합한다.

첫째, Random Fourier Features (RFF)를 사용한 RBF 커널 근사이다. OC-SVM의 핵심인 RBF 커널을 RFF를 통해 근사함으로써 커널 연산을 명시적인 특성 매핑으로 대체하고, 이를 통해 stochastic gradient descent (SGD)를 이용한 최적화가 가능해진다.

둘째, 딥 오토인코더와 OC-SVM의 공동 최적화이다. 오토인코더의 병목 계층(bottleneck layer)이 직접적으로 OC-SVM의 입력으로 사용되며, 두 구성 요소가 목적 함수를 공유함으로써 차원 감소가 이상 탐지 작업을 지원하는 방향으로 학습된다.

셋째, 그래디언트 기반 설명 방법의 적용이다. Kernel approximated OC-SVM에 그래디언트 기반 속성 방법(gradient-based attribution methods)을 적용하여, 입력 특성이 이상 탐지 결정에 어떻게 기여하는지 해석할 수 있게 한다. 이는 딥러닝 기반 이상 탐지에서 해석 가능성을 탐구한 최초의 연구로 알려져 있다.

### 기존 접근 방식과의 차별점

기존 연구와의 주요 차이점은 다음과 같다. 이전의 하이브리드 접근법들은 차원 감소와 이상 탐지를 별도로 수행하였으나, 본 논문의 AE-1SVM은 두 과정을 joint objective function을 통해 동시에 최적화한다. 또한 기존 연구들은 RFF를 입력 공간에 직접 적용하고 선형 SVM으로 처리하는 경우가 대부분이었으나, 본 논문에서는 커널 근사와 SGD 최적화를 OC-SVM에 함께 적용한다. 그리고 해석 가능성 측면에서, 기존 OC-SVM은 입력 공간과 커널 공간 사이의 암시적 매핑으로 인해 설명이 어렵다는 문제가 있었으나, 본 방법에서는 end-to-end 그래디언트를 통해 각 입력 특성의 기여도를 정량화할 수 있다.

## 3. 상세 방법 설명

### 전체 파이프라인 및 시스템 구조

AE-1SVM 모델의 아키텍처는 두 가지 주요 구성 요소로 이루어진다. 첫 번째 구성 요소는 입력 공간의 차원 감소 및 특성 표현을 위한 딥 오토인코더 네트워크이고, 두 번째 구성 요소는 support vectors와 margin을 기반으로 이상 예측을 수행하는 OC-SVM이다. OC-SVM의 RBF 커널은 Random Fourier Features를 통해 근사된다.

모델의 전체 흐름은 다음과 같다. 입력 데이터 x가 딥 오토인코더를 통과하여 병목 계층에서 저차원 표현 x를 얻는다. 이 잠재 표현이 RFF 매퍼로 전달되어 OC-SVM의 입력이 된다. 오토인코더의 디코더 부분은 입력 데이터의 재구성 값을 생성한다.

### 주요 구성 요소 및 역할

오토인코더 네트워크는 인코딩 계층(encoding layers)을 통해 입력 데이터를 저차원 잠재 공간으로 압축하고, 디코딩 계층(decoding layers)을 통해 원래 차원으로 복원한다. 병목 계층의 크기는 데이터셋에 따라 다르며, 실험에서는 다양한 설정이 사용되었다.

OC-SVM은 원점에서 이상치들을 분리하기 위한 초평면(hyperplane)을 학습한다. RFF는 명시적 특성 매핑을 통해 커널 트릭을 근사하며, 이를 통해 SGD를 이용한 최적화가 가능해진다.

Random Fourier Features 매퍼는 커널 근사를 위한 매핑 함수를 구현한다. 결합된 cosine과 sine 매핑을 사용하여 다음과 같이 정의된다.

$$z(x) = \sqrt{\frac{1}{D}}[cos(\omega_1^T x)...cos(\omega_D^T x) \quad sin(\omega_1^T x)...sin(\omega_D^T x)]^T$$

여기서 $\omega_1, \omega_2, ..., \omega_D$는 Gaussian 분포 $p(\omega) = \mathcal{N}(0, \sigma^{-2}\mathbf{I})$에서 독립적으로 추출된 가중치이다.

### 훈련 목표 및 손실 함수

AE-1SVM의 결합 목적 함수는 다음과 같이 정의된다.

$$Q(\theta, \mathsf{w}, \rho) = \alpha L(\mathbf{x}, \mathbf{x}') + \frac{1}{2}\|\mathsf{w}\|^2 - \rho + \frac{1}{\nu n}\sum_{i=1}^{n}\max(0, \rho - \mathsf{w}^T z(x_i))$$

이 목적 함수의 각 구성 요소는 다음과 같은 역할을 한다.

$L(\mathbf{x}, \mathbf{x}')$는 오토인코더의 재구성 손실로, 일반적으로 L2-norm 손실 $L(\mathbf{x}, \mathbf{x}') = \|\mathbf{x} - \mathbf{x}'\|\_2^2$로 정의된다. 이 손실은 입력 데이터와 재구성된 출력 사이의 차이를 최소화한다.

$\frac{1}{2}\|\mathsf{w}\|^2 - \rho + \frac{1}{\nu n}\sum_{i=1}^{n}\max(0, \rho - \mathsf{w}^T z(x_i))$는 kernel-approximated OC-SVM의 비제약 목적 함수이다. 여기서 $\mathsf{w}$는 커널 공간의 가중치 벡터이고, $\rho$는 원점에서 초평면까지의 거리를 결정하는 오프셋 파라미터이며, $\nu$는 정규화 파라미터로 데이터에서 이상치의 비율에 대한 상한선이다.

$\alpha$는 특징 압축과 SVM margin 최적화 사이의 트레이드오프를 조절하는 하이퍼파라미터이다.

### 학습 절차 및 알고리즘 흐름

학습은 SGD와 역전파(backpropagation)를 통해 결합 목적 함수를 동시에 최적화한다. 미니배치 gradient descent가 사용되며, 배치 크기는 데이터셋에 따라 16에서 1024 사이로 설정된다. Adam optimizer가 최적화 알고리즘으로 사용되며, 학습률은 데이터셋에 따라 0.001에서 0.01 사이로 조정된다.

### 그래디언트 계산 및 해석 가능성

OC-SVM margin 함수의 입력에 대한 그래디언트를 다음과 같이 유도한다. Margin 함수는 다음과 같이 정의된다.

$$g(x) = \sum_{j=1}^{D}\mathsf{w}\_j z_{\omega_j}(x) - \rho = \sqrt{\frac{1}{D}}\sum_{j=1}^{D}\big[\mathsf{w}\_j cos(\sum_{k=1}^{d}\omega_{jk}x_k) + \mathsf{w}\_{D+j} sin(\sum_{k=1}^{d}\omega_{jk}x_k)\big] - \rho$$

입력 차원 $x_k$에 대한 margin 함수의 그래디언트는 다음과 같이 계산된다.

$$\frac{\partial g}{\partial x_k} = \sqrt{\frac{1}{D}}\sum_{j=1}^{D}\omega_{jk}\big[-\mathsf{w}\_j sin(\sum_{k=1}^{d}\omega_{jk}x_k) + \mathsf{w}\_{j+D} cos(\sum_{k=1}^{d}\omega_{jk}x_k)\big]$$

이 그래디언트를 통해 각 입력 특성이 이상 탐지 결정에 얼마나 기여하는지 정량화할 수 있다. 그래디언트의 부호에 따른 해석은 다음과 같다. 그래디언트가 양수인 경우, 해당 차원의 특성 값이 정규 클래스의 하한선보다 작다는 것을 의미한다. 반면 그래디언트가 음수인 경우, 해당 특성 값이 정규 클래스의 범위를 초과한다는 것을 의미한다.

## 4. 실험 및 결과

### 데이터셋

실험은 하나의 합성 데이터셋과 다섯 개의 실제 데이터셋에서 수행되었다.

Gaussian 데이터셋은 고차원 대규모 데이터에서 방법의 성능을 보여주기 위해 설계되었다. 정규 샘플은 평균 0, 표준편차 $\sigma=1$인 정규분포에서 추출되고, 이상치 샘플은 $\sigma=5$인 정규분포에서 추출된다.

ForestCover는 ForestCover/Covertype 데이터셋에서 파생되었으며, 클래스 2를 정규 클래스로, 클래스 4를 이상치 클래스로 사용한다.

Shuttle는 Shuttle 데이터셋에서 파생되었으며, 클래스 2, 3, 5, 6, 7을 정규 샘플로, 클래스 1을 이상치로 사용한다.

KDDCup99는著名的 네트워크 침입 탐지 데이터셋으로, 오염 비율이 5%가 되도록 재구성되었다.

USPS는 우체국 손글씨 숫자 데이터셋으로, 숫자 '1'을 정규 데이터(950개), 숫자 '7'을 이상 데이터(50개)로 사용한다.

MNIST는 숫자 '4'를 정규 클래스로, 숫자 '0', '7', '9'를 이상치로 사용한다.

### 기준선 방법

비교를 위해 다양한 기준선 방법이 사용되었다. Raw 입력에 대해 직접 학습된 OC-SVM with RBF kernel, 분리된 오토인코더로 차원 감소 후 학습된 OC-SVM, Isolation Forest, Robust Deep Autoencoder (RDA), Deep Clustering Embeddings (DEC) 등이 포함되었다.

### 평가 지표

AUROC (Area Under Receiver Operating Characteristic curve)와 AUPRC (Area Under Precision-Recall curve)가 평가 지표로 사용되었다. AUROC는 모델의 전반적인 판별 능력을 측정하고, AUPRC는 불균형 데이터셋에서 방법들 간의 차이를 강조한다.

### 주요 정량적 결과

실험 결과에서 AE-1SVM은 대부분의 데이터셋에서 최고 또는 최고 수준의 성능을 달성하였다.

ForestCover 데이터셋에서 AE-1SVM은 AUROC 0.9485, AUPRC 0.1976을 달성하였다. 이는 Isolation Forest (AUROC 0.9396, AUPRC 0.0705)보다 AUROC 측면에서는 약간 높고, AUPRC 측면에서는 약 3배 높은 성능을 보였다.

Shuttle 데이터셋에서 AE-1SVM은 AUROC 0.9747, AUPRC 0.9483을 달성하였다. Isolation Forest가 AUROC에서 약간 앞서지만 (0.9816), AUPRC에서 AE-1SVM이 상당히 우월하다.

USPS 데이터셋에서 AE-1SVM은 AUROC 0.9926, AUPRC 0.8024로 대부분의 기준선을 능가하였다.

MNIST 데이터셋에서 AE-1SVM (AUROC 0.8119)에 convolutional autoencoder를 적용한 CAE-1SVM은 AUROC 0.8564를 달성하여 추가적인 개선을 보였다.

### 훈련 및 테스트 시간

훈련 시간 측면에서 AE-1SVM은 가장 큰 데이터셋인 ForestCover에서 다른 방법들보다 빠른 성능을 보였다. KDDCup99 전체 데이터셋에서 AE-1SVM은 약 200초 만에 유망한 결과를 얻었다. 테스트 시간에서 AE-1SVM은 Isolation Forest와 기존 OC-SVM보다 현저히 개선된 성능을 보였으며, 이는 실시간 환경에서의 적용 가능성을 시사한다.

### 그래디언트 기반 설명 검증

그래디언트 기반 설명 방법의 유효성은 USPS와 MNIST 이미지 데이터셋에서 검증되었다. 실험 결과, 정규 클래스의 밝은 픽셀이 나타나는 중앙 부분에서 양의 그래디언트가 집중되었고, 이미지 중앙 외부에서 음의 그래디언트가 집중되었다. 이는 모델이 해당 픽셀 값이 정규 클래스의 범위(검은색 수준)를 초과한다고 판단했음을 의미한다.

## 5. 강점, 한계

### 논문에서 뒷받침되는 강점

본 논문의 주요 강점은 다음과 같다.

End-to-end 학습을 통한 성능 향상: 분리된 차원 감소와 이상 탐지 학습 대신 joint training을 통해 AE-1SVM은 모든 시나리오에서 분리된 OC-SVM variants보다 우수한 성능을 달성하였다.

대규모 데이터 처리 능력: SGD와 RFF 커널 근사를 통해 기존 OC-SVM의 O(n²) 복잡성 문제를 극복하고, 대규모 데이터셋에서도 효율적으로 작동한다.

향상된 속도: 특히 테스트 시간에서 Isolation Forest와 기존 OC-SVM보다 현저히 빠른 응답 시간을达成하여 실시간 환경에 적합하다.

실용적 해석 가능성: 딥러닝 기반 이상 탐지에서 그래디언트 기반 속성 방법을 적용한 것은 중요한 기여로, 모델의 결정을 입력 특성 수준에서 설명할 수 있게 한다.

컨볼루션 오토인코더로의 확장 가능성: MNIST 실험에서 CAE-1SVM이 일반 오토인코더 기반 AE-1SVM보다 더 나은 성능을 보이며, 이미지 데이터로의 확장이 가능함을 시사한다.

### 한계, 가정 및 미해결 질문

하이퍼파라미터 의존성: RFF의 표준편차 $\sigma$, 정규화 파라미터 $\nu$, 트레이드오프 파라미터 $\alpha$ 등 여러 하이퍼파라미터의 조정이 필요하다. 저자들은 $\sigma=3.0$이 대부분의 데이터셋에서 만족스러운 결과를 낸다고 보고하였으나, 최적의 하이퍼파라미터 선택을 위한 체계적인 방법론은 제시되지 않았다.

RFF의 랜덤 특성: Random Fourier Features는 데이터에 독립적이므로, 여러 실행에서 결과가 달라질 수 있다. 저자들은 이를 명시적으로 다루지 않았으며, reproducibility를 위한 구체적인 시드 설정이나 앙상블 방법에 대한 논의가 부족하다.

비지도 학습 가정의 현실성: 실험 설정에서 전체 학습 데이터셋(정규와 이상 모두 포함)를 사용하여 모델을 학습한다. 그러나 실제 응용에서는 이상 데이터가 거의 없거나 전혀 없는 상황을 가정하는 경우가 많다.

커널 근사의 정확도: RFF는 RBF 커널을 근사하므로, 충분한 수의 랜덤 특성이 필요할 수 있다. 저자들은 RFF 차원 D에 대한 체계적인 분석 없이 실험 설정에서 50에서 1000 사이의 값을 사용하였다.

새로운 이상치에 대한 처리: 밀도 추정 기반 방법들과 마찬가지로, 새로운 형태의 이상치가 많은 경우 높은 밀도가 할당되어 거짓 음성(false negative)을 생성할 가능성이 있다.

### 비판적 해석

전반적으로 본 논문은 OC-SVM과 딥러닝의 통합이라는 중요한 연구 방향을 다루고 있으며, joint training의 효과와 해석 가능성에 대한 기여는 의미 있다. 그러나 몇 가지 점에서 비판적 검토가 필요하다.

기준선 선택에 있어 일부 기존 최첨단 방법들(예: GAN 기반 이상 탐지)과의 비교가 부족하다. 또한 실제 산업 응용에서 중요한 모델 배포 및 업데이트 측면의 논의가 없다.

## 6. 결론

### 주요 기여 사항 요약

본 논문은 딥 오토인코더와 kernel-approximated OC-SVM을 결합한 AE-1SVM 모델을 제안하였다. 주요 기여 사항은 다음과 같다.

End-to-end 학습을 통해 차원 감소와 이상 탐지를 동시에 최적화하여, 분리된 학습 방식보다 우수한 성능을 달성하였다. Random Fourier Features를 사용한 RBF 커널 근사를 통해 SGD 기반 최적화를 가능하게 하고, 대규모 고차원 데이터셋에서의 확장성 문제를 해결하였다. Kernel-approximated OC-SVM에 그래디언트 기반 속성 방법을 적용하여, 딥러닝 기반 이상 탐지에서의 해석 가능성을 탐구하였다.

### 실제 적용 및 향후 연구의 가능성

실시간 이상 탐지가 필요한 산업 응용 분야에서 AE-1SVM의 활용 가능성이 크다. 네트워크 침입 탐지, 제조 결함 탐지, 사기 탐지 등 빠른 응답이 요구되는 영역에서 훈련 및 테스트 시간 측면의 개선된 성능이 유리하게 작용할 수 있다.

향후 연구 방향으로는 RFF의 최적 구성에 대한 이론적 분석, 다양한 이상 탐지 도메인에서의 평가, 온라인 학습 설정으로의 확장, 앙상블 방법을 통한 안정성 향상, Semi-supervised 설정에서의 연구 등이 제안될 수 있다.
